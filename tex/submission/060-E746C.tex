\section{Introduction}
\subsection{Background}
Nowadays, many datasets can be naturally represented as graphs, and it is often of interest to study
the underlying structure of these graphs. One important example is the problem of community detection, 
where we want to find clusters of nodes that are more densely connected to each other than to the
rest of the graph. The stochastic block model (SBM) is a popular random graph model that captures
this community structure, and has been widely studied in the literature. It randomly assigns each 
vertex to one of $k$ communities, and then generates edges independently between vertices with 
probabilities that depend on their community memberships. This model is simple yet rich enough to
capture many real-world phenomena, and has been used as a benchmark for testing community detection.
In this paper, we will focus on community detection under the balanced, two-community SBM, which
we define below.
\begin{definition}[Balanced, two-community SBM]
A graph $G \sim \mathcal{G}(n, p, q)$ is a balanced, two-community stochastic block model with 
$V = [n]$ and uniformly random label vector $\bm{x} \in \{\pm 1\}^n$ satisfying $\bm{1}^\top \bm{x} = 0$
if, for each pair of vertices $i, j \in [n]$, we have
\[\mathbb{P}((i,j) \in E) = \begin{cases}
p & \text{if } x_i = x_j, \\
q & \text{if } x_i \neq x_j.
\end{cases}\] 
\end{definition}
Usually, we have $p > q$, so that vertices in the same community are more likely to be connected 
than those in different communities, demonstrating the community structure.

In practice, however, real data never perfectly follows any theoretical model. Randomness or corruption
in the data can lead to nontrivial deviations, so it is important to design algorithms that are 
robust to such perturbations. In the context of community detection, this means that we want to
develop algorithms that can still recover the underlying community structure in some capacity,
even under these contaminations. Thus, in this essay, we will study algorithms that can achieve
robust recovery of the community labels under various perturbation models. In particular, 
we focus on algorithms based on semidefinite programming (SDP), which has been used to tackle a wide
range of robustness problems.

\subsection{Evaluating recovery}
In our setting, we can measure the recovery quality by the correlation between the true community 
labels and the estimated community labels. Suppose $\hat{\bm{x}}$ is the estimated community label 
vector, then we can define the correlation as the inner product
$\langle \hat{\bm{x}}, \bm{x} \rangle / n$. If we guess the labels randomly, then the expected 
correlation is zero. We define the idea of recovery quality as follows, following \cite{Abbe23}.
\begin{definition}[Recovery quality]
Given a balanced, two-community SBM $G$ with true community labels $\bm{x}$, an estimated community
label vector $\hat{\bm{x}}$ achieves:

- \textbf{exact recovery,} if $|\langle \hat{\bm{x}}, \bm{x} \rangle| / n = 1$ with high probability;

- \textbf{almost exact recovery,} if $|\langle \hat{\bm{x}}, \bm{x} \rangle| / n = 1 - o(1)$ 
with high probability;

- \textbf{weak recovery,} if $|\langle \hat{\bm{x}}, \bm{x} \rangle| / n > \alpha$ for some 
$\alpha > 0$ with high probability.
\end{definition}
In this paper, when we say ``with high probability'', we mean that the probability goes to $1$ as
the number of vertices $n$ goes to infinity.

\subsection{Semidefinite programming}
Semidefinite programming (SDP) is a powerful tool in optimization. It is an extension of linear
programming where we optimize a linear function over the set of positive semidefinite matrices 
satisfying some set of linear constraints. Formally, an SDP can be written in the following standard form:
\begin{align*}
\text{minimize}\quad &\langle \bm{C}, \bm{X} \rangle\\
\text{subject to}\quad &\langle \bm{A}_i, \bm{X} \rangle = b_i \text{ for } i = 1, \ldots, m,\\
&\bm{X} \succeq 0,
\end{align*}
where $\langle \cdot, \cdot \rangle$ denotes the matrix inner product (sum of entrywise products), 
$\bm{C}, \bm{A}_1, \ldots, \bm{A}_m$ are given symmetric matrices, $b_1, \ldots, b_m$ are given 
scalars, and $\bm{X}$ is the variable matrix we want to optimize over.

In our context, we will usually have $\bm{C} = \bm{A}_G - \bm{K}$, where $\bm{A}_G$ is the adjacency
matrix of the graph $G$, and $\bm{K}$ is some correction matrix that depends on the model parameters
and the observed graph. The constraints will typically enforce that $\bm{X}$ has diagonal entries 
all equal to $1$, and sometimes other linear constraints. We will use the following notations.
\begin{definition}[Semidefinite programming notations]
Denote the set of positive semidefinite matrices with diagonal entries equal to $1$ by
\[
\mathcal{PSD}_1(n) = \{\bm{X} \in \mathbb{R}^{n \times n} : \bm{X} \succeq 0, \bm{X}_{ii} = 1 \text{ for all } i \in [n]\}.
\]
Write $\mathcal{A}$ for the set of matrices satisfying some linear constraints, i.e.
\[\mathcal{A} = \{\bm{X} \in \mathbb{R}^{n \times n} : \langle \bm{A}_i, \bm{X} \rangle = b_i \text{ for } i = 1, \ldots, m\}.\]
Given a matrix $\bm{M} \in \mathbb{R}^{n \times n}$, we define 
\[\mathcal{SDP}_\mathcal{A}(\bm{M}) := \max_{\bm{X} \in \mathcal{PSD}_1(n) \cap \mathcal{A}} \langle \bm{M}, \bm{X} \rangle.\]
In particular, if there are no extra constraints, we omit the subscript $\mathcal{A}$ and simply 
write $\mathcal{SDP}(\bm{M})$.
\end{definition}

\section{Existing robustness models in community detection}
\label{sec:existing.robustness}
\subsection{Monotone adversary, exact recovery}
One of the earlier works on robust community detection is by Feige and Kilian \cite{FK01} in 2001.
They considered the following monotone adversary model, and came up with an SDP algorithm that
achieves exact recovery against this adversary.
\begin{definition}[Monotone adversary model]
\label{def:monotone}
Let $G$ be a balanced, two-community SBM. A monotone adversary can look at the realised graph $G$
and the community labels $x$ of the vertices, and they are allowed to make the following changes:
\begin{enumerate}
    \item[(a)] Add edges between vertices in the same community.
    \item[(b)] Remove edges between vertices in different communities.
\end{enumerate}
\end{definition}
It may seem that the monotone adversary can only make the problem easier, since they are either:
(a), strengthening the community structure; or (b), removing inter-community connections. However,
certain naive algorithms already fail under this model.

\subsubsection{Naive example: common neighbors algorithm}
Consider the following naive algorithm taking in two nodes, and outputting whether they are in the
same community or not.
\begin{algorithm}[Common neighbors algorithm]
\label{alg:degree}
\leavevmode
\begin{enumerate}
  \item[0.] Take inputs: graph $G$ on $n$ vertices; $i, j \in [n]$: the two vertices to investigate;
  $d$: threshold for degree comparison.
  \item[1.] Count the number of common neighbors $n_{ij}$ of $i$ and $j$.
  \item[2.] Output ``same community'' if $n_{ij}/n \geq r(p,q)$, and ``different communities'' otherwise.
\end{enumerate}
\end{algorithm}
The $r(p,q)$ threshold is a constant that depends on $p$ and $q$. The idea behind the algorithm is
that, for $i$ and $j$ in the same community, the expected number of common neighbors is 
\[
\left(\frac{n}{2} - 2\right) p^2 + \frac{n}{2} q^2 \approx \frac{p^2 + q^2}{2} n,
\]
where the first term counts their common neighbors in their community, and the second term counts
their common neighbors in the other community. On the other hand, for $i$ and $j$ in different
communities, the expected number of common neighbors is
\[
(n-2) pq \approx (pq)n.
\]
Since $p^2 + q^2 > 2pq$ for $p > q$, we can choose $r(p,q) \in (pq, (p^2 + q^2)/2)$, so the algorithm
works with high probability.

Clearly, the algorithm above relies on the exact values of $p$ and $q$ heavily. The monotone adversary
can easily break this. For example, it can add all the edges in both communities. Then, for any two 
vertices in different communities, their expected number of common neighbors is $(n-2) q \approx qn$.
To combat this, we might want to instead choose $r(p,q) \in (q, (p^2 + q^2)/2)$, but this can be 
impossible, as $q$ can be larger than $(p^2 + q^2)/2$ when $p$ and $q$ are small. Thus, the common 
neighbors algorithm is not robust to the monotone adversary.

\subsubsection{SDP-based robust algorithm}
Now, we consider the following SDP problem. Consider the following linearly constrained set:
\[
\mathcal{A} = \{\bm{X} \in \mathbb{R}^{n \times n} : \sum_{i,j \in [n]} \bm{X}_{ij} = 0\}.
\]
Then, our SDP of interest is $\tilde{h}(G) = \mathcal{SDP}_\mathcal{A}(\bm{A}_G)$, which can be 
written explicitly as
\begin{align*}
\tilde{h}(G) = \mathrm{maximize}\quad &\langle \bm{A}_G, \bm{X} \rangle \\
\mathrm{subject}\ \mathrm{to}\quad &\bm{X} \succeq 0,\\
&\bm{X}_{ii} = 1 \text{ for all } i \in [n],\\
&\sum_{i,j \in [n]} \bm{X}_{ij} = 0.
\end{align*}

To gain some intuition behind this SDP, let us consider a set of balanced community labels 
$\hat{x} \in \{\pm 1\}^n$. It is easy to see that the matrix $\hat{\bm{X}} := \hat{x} \hat{x}^\top$ 
is a feasible solution to the above SDP. Plugging in $\hat{\bm{X}}$ into the objective, we have
\begin{align*}
\langle \bm{A}_G, \hat{\bm{X}} \rangle &= \sum_{i,j \in [n]} \bm{A}_{G,ij} \hat{x}_i \hat{x}_j
= \sum_{i,j \in E} \hat{x}_i \hat{x}_j\\
&= \sum_{i,j \text{ in same community}} 1 + \sum_{i,j \text{ in different communities}} (-1)\\
&= 2\times \text{number of edges within communities} - 2\times \text{number of edges between communities}\\
&= 2|E| - 4\times\text{number of edges between communities}.
\end{align*}

Thus, if we maximize the above SDP over all balanced community labels, we are essentially trying to
minimize the number of edges between communities. By making the SDP relaxation, we make the problem
tractable. Writing $h(G) := |E|/2 - \tilde{h}(G)/4$, $b(G) := \text{number of edges between communities}$,
then we see that $h(G) \leq b(G)$ is an SDP lower bound on the number of edges between communities. 
In fact, we often have $h(G) = b(G)$, by the following result.
\begin{lemma}[Lemma 19 of \cite{FK01}]
\label{lem:SBM.hb}
Let $G$ be a balanced, two-community SBM with edge probabilities $p$ and $q$ as defined above. 
If $p - q \geq C \sqrt{p \log n / n}$ for a sufficiently large constant $C$, then with
probability at least $1 - 2 n^{-5}$, we have $h(G) = b(G)$.
\end{lemma}
Furthermore, it can be shown that modifying one edge of $G$ preserves the equality $h(G) = b(G)$ 
with high probability.
\begin{lemma}[Lemma 20 of \cite{FK01}]
\label{lem:modSBM.hb}
Let $G$ be a graph satisfying $h(G) = b(G)$. Let $G'$ be the graph obtained by modifying one edge of
$G$ (adding a new edge or removing an existing one). Then, with probability at least
$1 - n^{-1 + o(1)}$, we have $h(G') = b(G')$ for all possible modifications.
\end{lemma}
This inspires the following algorithm.
\begin{algorithm}[SDP-based exact recovery under monotone adversary, \cite{FK01}]
\label{alg:monotone.SDP}
\leavevmode
\begin{enumerate}
  \item[0.] Take input: graph $G$ on $n$ vertices.
  \item[1.] For $i \in [n]\backslash \{1\}$, let $G_i$ be the graph obtained by ``toggling'' the edge 
  $(1,i)$: i.e. if $G$ contains the edge $(1,i)$, remove it; otherwise, add it.
  \item[2.] Writing $S_1 = \{1\} \cup \{i \in [n]\backslash \{1\}: h(G_i) = h(G)\}$,
  output the community labels $\hat{\bm{x}}$ as follows:
  \[
  \hat{\bm{x}}_i = \begin{cases}
1 & \text{if } i \in S_1,\\
-1 & \text{if } i \notin S_1.
\end{cases}
  \]
\end{enumerate}
\end{algorithm}
We can show that this algorithm recovers the true community labels with high probability, 
and is robust to any monotone adversary.
\begin{theorem}[Exact recovery under monotone adversary, Theorem 5 of \cite{FK01}] 
\label{thm:monotone}
Let $G$ be a balanced, two-community SBM with intra-community edge probability $p$ and 
inter-community edge probability $q$, and $C$ be a sufficiently large constant. 
If $p - q \geq C \sqrt{p \log n / n}$, then with high probability, the above algorithm
recovers the true community labels, and is robust to any monotone adversary.
\end{theorem}
\begin{proof}
We work under the event that $h(G) = b(G)$, and that for all $i \in [n]\backslash \{1\}$, 
$h(G_i) = b(G_i)$. By Lemmas \ref{lem:SBM.hb} and \ref{lem:modSBM.hb}, this event occurs with high 
probability.

If $1$ and $i$ are in the same community, then the edge $(1,i)$ is an intra-community edge, and
will not affect the inter-community edge count $b$, so we have $b(G_i) = b(G)$. On the other hand, 
if $1$ and $i$ are in different communities, then the edge $(1,i)$ is indeed an inter-community edge,
so $b(G_i) = b(G) \pm 1$, so $h(G_i) \neq h(G)$. Hence, we've shown exact recovery.

To show robustness against the monotone adversary, it suffices to show the following:
\begin{lemma}[Monotone modifications preserve $h = b$]
\label{lem:monotone.robust}
Let $G$ be a balanced SBM satisfying $h(G) = b(G)$. Let $G'$ be the graph obtained by making one 
monotone modification to $G$ (i.e. adding an edge between vertices in the same community, or removing
an edge between vertices in different communities). Then, we have $h(G') = b(G')$.
\end{lemma}
Note that this lemma is an exact statement, so we can make arbitrary many monotone modifications.
Combining this lemma with Lemmas \ref{lem:SBM.hb} and \ref{lem:modSBM.hb}, we have that $h(G) = b(G)$,
and $h(G_i) = b(G_i)$ for all $i \in [n]\backslash \{1\}$ still holds with high probability after
any number of monotone modifications, and thus the algorithm still works.

To prove this lemma, the following three properties of $h$ will be helpful:

- \textbf{Lower bound of $b$:} Since $b(G)$ is the value of the objective at a feasible solution to 
the SDP, we have
\[h(G) \leq b(G).\]

- \textbf{Increasing:} Writing $G^+$ for the graph obtained by adding an edge $(u,v)$ to $G$, 
and $\bm{X}^+$, $\bm{X}$ for the optimal solutions to the SDP for $G^+$ and $G$ respectively, we have
\begin{align*}
h(G^+) - h(G) &= \left(\frac{|E(G^+)|}{2} - \frac{\tilde{h}(G^+)}{4}\right) 
- \left(\frac{|E(G)|}{2} - \frac{\tilde{h}(G)}{4}\right)\\
&= \frac{1}{2} - \frac{1}{4} \left(\langle \bm{A}_{G^+}, \bm{X}^+ \rangle - \langle \bm{A}_G, \bm{X} \rangle\right)\\
&\geq \frac{1}{2} - \frac{1}{4} \left(\langle \bm{A}_{G^+}, \bm{X} \rangle - \langle \bm{A}_G, \bm{X} \rangle\right)\\
&= \frac{1}{2} - \frac{1}{4} (\bm{X}_{uv} + \bm{X}_{vu}) \geq 0,
\end{align*}
where we used the fact that $|\bm{X}|_\infty \leq 1$. So $h(G^+) \geq h(G)$.

- \textbf{Bounded increments:} Proceeding in exactly the same way as above, we have
\begin{align*}
h(G^+) - h(G) &= \frac{1}{2} - \frac{1}{4} \left(\langle \bm{A}_{G^+}, \bm{X}^+ \rangle 
- \langle \bm{A}_G, \bm{X} \rangle\right)\\
&\leq \frac{1}{2} - \frac{1}{4} \left(\langle \bm{A}_{G^+}, \bm{X^+} \rangle - \langle \bm{A}_G, \bm{X^+} \rangle\right)\\
&= \frac{1}{2} - \frac{1}{4} (\bm{X}_{uv} + \bm{X}_{vu}) \leq 1,
\end{align*}
where we again used the fact that $|\bm{X}|_\infty \leq 1$. So $h(G^+) \leq h(G) + 1$.

Now, we are ready to prove the lemma. We call the bisection of $G$ that achieves the optimal value
$b(G)$ the \emph{optimal bisection} of $G$, and the two communities created by this bisection the 
\emph{optimal communities} of $G$.

If $G^+$ is obtained by adding an edge between vertices in the
same community, the number of edges between any two balanced communities can only increase. 
In particular, the number of edges between the optimal communities of $G$ remains the same, 
so this bisection remains optimal for $G^+$. Thus, $b(G^+) = b(G)$.
By the lower bound property, we have $h(G^+) \leq b(G^+) = b(G) = h(G)$, and by the increasing
property, we have $h(G^+) \geq h(G)$. Hence, all the inequalities are equalities, and we have 
$h(G^+) = b(G^+)$.

On the other hand, if $G^-$ is obtained by removing an edge between vertices in different communities,
the number of edges between any two balanced communities can at most decrease by one.
In particular, the number of edges between optimal communities of $G$ also decreases by one, so
this bisection remains optimal for $G^-$. Thus, $b(G^-) = b(G) - 1$.
By the lower bound property, we have $h(G^-) \leq b(G^-) = b(G) - 1 = h(G) - 1$, and by the bounded
increments property, we have $h(G^-) \geq h(G) - 1$. Hence, all the inequalities are equalities, 
and we have $h(G^-) = b(G^-)$.

Therefore, $h(G) = b(G)$ is preserved under any monotone modification, which proves the lemma, and thus the theorem.
\end{proof}
\begin{remark}
Even though monotone adversaries are quite limited in their power, the above result is still
interesting in that it does not require any knowledge of the parameters $p$ and $q$. Furthermore,
it highlights the fundamental reason why SDP-based algorithms can be robust: they are increasing, 
but have bounded increments, so small modifications to the graph does not change the output by much.
In this particular case, by leveraging the fact that the optimal objective value exactly correlates
with an important graph statistic (the minimal number of edges between communities) with high probability,
we are able to strengthen the robustness guarantee to hold for any number of monotone modifications.
In our cases, we will not be able to achieve such a strong guarantee, but this ``bounded difference''
property of SDP will still be crucial. For example, in \cref{sec:MS16}, we will relate 
$\mathcal{SDP}(\bm{A}_G^\text{cent})$ to the SDP value of some Gaussian matrix, which will be the
central component allowing the robust algorithm to work.
\end{remark}

\subsection{Edge-robust weak recovery}
\label{sec:MS16}
Now let us consider a different setting, where we want to instead achieve weak recovery with the
minimal amount of community structure. We first introduce the following formulation of the SBM, 
which is more convenient for our analysis.
\begin{definition}[$d,\epsilon$-SBM formulation]
A graph $G \sim \mathcal{G}(n, d, \epsilon)$ is a balanced, two-community stochastic block model graph
with $V = [n]$ and uniformly random label vector $\bm{x} \in \{\pm 1\}^n$ satisfying $\bm{1}^\top \bm{x} = 0$,
if for each pair of vertices $i, j \in [n]$, we have
\[\mathbb{P}((i,j) \in E) = \begin{cases}
\frac{d}{n}(1+\frac{\epsilon}{2}) & \text{if } x_i = x_j, \\
\frac{d}{n}(1-\frac{\epsilon}{2}) & \text{if } x_i \neq x_j.
\end{cases}\] 
\end{definition}
Note that $d$ is the average degree of the graph, and $\epsilon$ controls the strength of the community structure.
These parameters are assumed to be known constants, and we are interested in the case where $n$ could
be very large. Hence, we are working in a sparse regime compared to the previous section, 
where $p$ and $q$ are constants. 

How much community structure do we really need to achieve weak recovery? It has been shown that 
\cite{MNS15}, in the sparse regime where $d$ is constant, weak recovery is possible if and only if 
\[\lambda := \frac{\epsilon \sqrt{d}}{2} > 1.\]
This threshold is known as the Kesten-Stigum threshold. There are various algorithms that can achieve
weak recovery above the Kesten-Stigum threshold, but they are not known to be robust against edge
perturbations. In \cite{MS16}, Montanari and Sen proposed a simple SDP-based algorithm that can achieve
weak recovery almost at the Kesten-Stigum threshold, and is automatically robust against edge perturbations.

Let us first introduce the notion of the centered adjacency matrix.
\begin{definition}[Centered adjacency matrix]
Given a graph $G$, let $\bm{A}_G$ be its adjacency matrix, and define $\bm{A}_G^{\mathrm{cent}} = \bm{A}_G - \mathbb{E}[\bm{A}_G]$.
For $G \sim \mathcal{G}(n, d, \epsilon)$, we have $\mathbb{E}[\bm{A}_G] = \frac{d}{n}\bm{1}\bm{1}^\top$.
\end{definition}

In their paper, Montanari and Sen looked at $\mathcal{SDP}(\bm{A}_G^{\mathrm{cent}})$, the basic
SDP problem on the centered adjacency matrix, and proposed the following algorithm.
\begin{algorithm}[SDP Algorithm from \cite{MS16}]
\label{alg:MS16}
\leavevmode
\begin{enumerate}
    \item Sample edges of $G$ independently with probability $(1 + n^{-1/2})^{-1}$ to obtain the sampled edge set $E_\mathrm{sample} \subset E$, and then compute the evaluation edge set $E_\mathrm{eval} = E \setminus E_\mathrm{sample}$.
    \item Construct the subgraphs $G_\mathrm{sample} = (V, E_\mathrm{sample})$ and $G_\mathrm{eval} = (V, E_\mathrm{eval})$.
    \item Compute an optimiser $\bm{X}^*$ of $\mathcal{SDP}(\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}})$, i.e., $\langle \bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}, \bm{X}^* \rangle = \mathcal{SDP}(\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}})$.
    \item Compute the eigenvectors $\bm{v}^{(1)}, \ldots, \bm{v}^{(n)}$ of $\bm{X}^*$.
    \item For each $i,j \in [n]$, define $\hat{\bm{x}}^{(i,j)} \in \{-1, 0, 1\}^{n}$ as follows:
    \[\hat{\bm{x}}^{(i,j)}_k = \begin{cases}
      \operatorname{sign}(\bm{v}^{(i)}_k) & \text{if } |\bm{v}^{(i)}_k| \geq |\bm{v}^{(i)}_j|, \\
      0 & \text{otherwise}.
    \end{cases}\]
    \label{alg:MS16.rounding}
    \item Find $(i^*, j^*) = \arg\max_{i,j \in [n]} (\hat{\bm{x}}^{(i,j)})^\top \bm{A}_{G_\mathrm{eval}}^{\mathrm{cent}} \hat{\bm{x}}^{(i,j)} = \sum_{a,b \in E_\mathrm{eval}}\hat{\bm{x}}^{(i,j)}_a \cdot \hat{\bm{x}}^{(i,j)}_b$.
    \label{alg:MS16.selection}
    \item Output $\hat{\bm{x}}^\mathrm{SDP}(G) = \hat{\bm{x}}^{(i^*,j^*)}$.
\end{enumerate}
\end{algorithm}
We have the following weak recovery rate and edge robustness result for Algorithm \ref{alg:MS16}.
\begin{theorem}[Weak Recovery for Algorithm \ref{alg:MS16}]\label{thm:MS16.recovery}
Let $G \sim \mathcal{G}(n, d, \epsilon)$ and assume, for some $\delta > 0$, that $\lambda = \epsilon\sqrt{d}/2 \geq 1 + \delta$.
Then, there exists $\eta = \eta(\delta) > 0$ and $d_* = d_*(\delta) > 0$ such that, for all $d \geq d_*$,
the output of Algorithm \ref{alg:MS16}, $\hat{\bm{x}}^\mathrm{SDP}(G)$, satisfies
\[\mathbb{P}\left(\frac{1}{n} |\langle \hat{\bm{x}}^\mathrm{SDP}(G), x \rangle| \geq \eta \right) \geq 1 - Ce^{-\sqrt{n}/C}.\]
Furthermore, this algorithm is robust against $o(n\sqrt{d})$ edge perturbations, in the sense that
if we modify $G$ by adding or removing $o(n\sqrt{d})$ edges to obtain a new graph $G'$, then 
\end{theorem}
This will be the first main result we will focus on later in this essay in \cref{sec:weak.recovery}. 
For now, we will just give a high-level overview of the algorithm and the proof ideas.

The main idea behind the proof is that there exists some Gaussian random matrix $\bm{B}(\lambda)$, 
such that $\mathcal{SDP}(\bm{A}_{G}^{\mathrm{cent}}/\sqrt{d})$ is close to 
$\mathcal{SDP}(\bm{B}(\lambda))$, and $\mathcal{SDP}(\bm{A}_{G}^{\mathrm{cent}}/\sqrt{d}
- (\lambda / n)\bm{x}\bm{x}^\top)$ is close to $\mathcal{SDP}(\bm{B}(0))$. There is a phase
transition result for the SDP value of $\bm{B}(\lambda)$ at $\lambda = 1$, so we can bound
$\mathcal{SDP}(\bm{A}_{G}^{\mathrm{cent}}/\sqrt{d})$ and $\mathcal{SDP}(\bm{A}_{G}^{\mathrm{cent}}/\sqrt{d}
- (\lambda / n)\bm{x}\bm{x}^\top)$ apart. This is known as the \emph{push-out effect},
which guarantees that the optimizer to $\mathcal{SDP}(\bm{A}_{G}^{\mathrm{cent}}/\sqrt{d})$ 
must contain nontrivial information about $\bm{x}$. Then, we can show that the rounding procedure in 
Step \ref{alg:MS16.rounding} and the selection procedure in Step \ref{alg:MS16.selection} can 
extract this information to produce an estimator $\hat{\bm{x}}^\mathrm{SDP}(G)$ that 
achieves weak recovery.

If we allow a small number of edge perturbations, then by the bounded differences property of SDP, 
the two SDP values, $\mathcal{SDP}(\bm{A}_{G}^{\mathrm{cent}}/\sqrt{d})$ and
$\mathcal{SDP}(\bm{A}_{G}^{\mathrm{cent}}/\sqrt{d} - (\lambda / n)\bm{x}\bm{x}^\top)$, only undergo
small changes and remain bounded apart, so the rest of the proof still goes through.

\begin{remark}
Note that in the proof here, we again leveraged the ``bounded differences'' property of SDP, explicitly
in the edge robustness part, and also implicitly in the node robustness part where we relate SDP 
values of the observed graph to those of some Gaussian random matrices. Then, using an additional SDP
phase transition result for the Gaussian random matrices, we get the desired recovery and robustness
guarantees. This mirrors the proof structure of the exact recovery under monotone adversary before,
where we also leveraged the ``bounded differences'' property of SDP, and a different property
of the SDP value to get recovery and robustness guarantees.
\end{remark}

\subsection{Degree heterogeneity}
So far, we have only considered SBMs where all nodes have the same expected degree. In practice, 
this is not always the case. We can consider the more general degree-corrected SBM (DCSBM) model, 
where each node is given an additional expected degree parameter, and the edge probabilities are
determined by both the community labels and the expected degree parameters of the endpoints.
\begin{definition}[Balance, two-community, degree-corrected SBM]
A graph $G \sim \mathcal{G}(n, p, q, \bm{\theta})$ is a degree-corrected SBM graph with $V = [n]$, 
uniformly random label vector $\bm{x} \in \{\pm 1\}^n$ satisfying $\bm{1}^\top \bm{x} = 0$,
and node degree parameters $\bm{\theta} \in \mathbb{R}_+^n$ satisfying 
\[
\sum_{i: \bm{x}_i = 1} \bm{\theta}_i = \sum_{i: \bm{x}_i = -1} \bm{\theta}_i =: g,
\]
if, for each pair of vertices $i, j \in [n]$, we have
\[\mathbb{P}((i,j) \in E) = \begin{cases}
\bm{\theta}_i \bm{\theta}_j p & \text{if } \bm{x}_i = \bm{x}_j, \\
\bm{\theta}_i \bm{\theta}_j q & \text{if } \bm{x}_i \neq \bm{x}_j.
\end{cases}\]
\end{definition}
This definition can be extended to multiple unbalanced communities, but for consistency with previous
sections, we will only consider the balanced, two-community case here. The results we will discuss
are originally proven in the more general setting, but the balanced, two-community case is already
sufficient to illustrate the main ideas.

Chen et al. \cite{CLX18} developed a recovery algorithm for the DCSBM based on the SDP relaxation
of modularity maximization \cite{Newman06}. In particular, define the following linearly
constrained set:
\[
\mathcal{A} = {\bm{X} \in \mathbb{R}^{n\times n}: \bm{X}_{ij} \in [0,1], \forall i,j \in [n]}.
\]
Then, the SDP they considered is $\mathcal{SDP}_\mathcal{A}(\bm{A}_G - \lambda \bm{d}\bm{d}^\top)$,
where $\bm{d}$ is the degree vector of $G$, and $\lambda$ is a tuning parameter.

Define the quantities $H_\pm$ as follows:
\[
H_+ = p \sum_{i: \bm{x}_i = 1} \bm{\theta}_i + q \sum_{i: \bm{x}_i = -1} \bm{\theta}_i, \quad
H_- = p \sum_{i: \bm{x}_i = -1} \bm{\theta}_i + q \sum_{i: \bm{x}_i = 1} \bm{\theta}_i.
\]
Then, we see that the expected degree of a node $i$ is $\bm{\theta}_i H_{\bm{x}_i}$.
In our balanced, two-community case, we have $H_+ = H_- = g(p + q) =: H$.

Chen et al. \cite{CLX18} proved the following technical result demonstrating the correlation between
the optimizer of the above SDP and the true community labels.
\begin{theorem}[Modularity SDP for DCSBM, Corollary 3.1 of \cite{CLX18}]
\label{thm:CLX18}
Suppose that $p > q > 0$, and there exists $\delta > 0$ such that the tuning parameter $\lambda$ satisfies
\[
\frac{q + \delta}{H^2} \leq \lambda \leq \frac{p - \delta}{H^2}.
\]
Then, writing $\bm{X} = \bm{x}\bm{x}^\top$, with probability $0.99 - 2(e/2)^{-2n}$, any optimizer 
$\hat{\bm{X}}$ of $\mathcal{SDP}_\mathcal{A}(\bm{A}_G - \lambda \bm{d}\bm{d}^\top)$ satisfies
\[
\|\hat{\bm{X}} - \bm{X}\|_{1, \bm{\theta}} := 
\sum_{i, j \in [n]}\bm{\theta}_i\bm{\theta}_j|\hat{\bm{X}}_{ij} - \bm{X}_{ij}| 
\leq \frac{C}{\delta}(1 + \lambda \sigma)(\sqrt{n\sigma}+n),
\]
where $\sigma := 2g^2(p + q)$, and $C$ is a universal constant.
\end{theorem}
To very briefly explain the proof idea, we define the expected degree vector $\bm{f} = \mathbb{E}[\bm{d}]$.
We consider $\langle \bm{A}_G - \lambda \bm{d}\bm{d}^\top, \hat{\bm{X}} - \bm{X} \rangle$, which is 
non-negative by the optimality of $\hat{\bm{X}}$. We can decompose this quantity as follows:
\[
\langle \bm{A}_G - \lambda \bm{d}\bm{d}^\top, \hat{\bm{X}} - \bm{X} \rangle
= \langle \mathbb{E}[\bm{A}_G] - \lambda \bm{f}\bm{f}^\top, \hat{\bm{X}} - \bm{X} \rangle 
+ \langle \bm{A}_G - \mathbb{E}[\bm{A}_G], \hat{\bm{X}} - \bm{X} \rangle
+ \lambda \langle \bm{f}\bm{f}^\top - \bm{d}\bm{d}^\top, \hat{\bm{X}} - \bm{X} \rangle.
\]
Using the conditions in the theorem, we can show that the first term is upper bounded by
$-\delta \|\hat{\bm{X}} - \bm{X}\|_{1, \bm{\theta}}$. The second and third terms can be controlled
using Grothendieck's inequality, which is a concentration result relying on the fact that 
$\hat{\bm{X}} \in \mathcal{PSD}_1$. Indirectly, this is yet again using a ``bounded 
differences'' property, this time of the inner product rather than the SDP itself. 
Rearranging the terms, we get the desired inequality.

With this technical result, we have the following recovery corollary.
\begin{corollary}[Recovery for DCSBM, Theorem 3.2 of \cite{CLX18}]
With all the assumptions in \cref{thm:CLX18}, there exists a clustering algorithm producing an 
estimator $\hat{\bm{x}} \in \{\pm 1\}^n$ such that, with probability $0.99 - 2(e/2)^{-2n}$,
\[
\min \left\{\sum_{i: \bm{x}_i \neq \hat{\bm{x}}_i} \bm{\theta}_i, \sum_{i: \bm{x}_i = \hat{\bm{x}}_i} \bm{\theta}_i \right\} 
\leq \frac{C}{\delta} \left(\frac{n}{g} + 2\sqrt{np}\right),
\]
where the minimum is taken over a possible global sign flip.
\end{corollary}
Note that this notion of recovery is slightly different from the ones we have seen before, since
the error is weighted by the node degree parameters. But we see that this weighted number of 
misclassifications is at most a fraction of $n$, which is a ``weak recovery'' style guarantee.

This corollary can be interpreted as a robustness result in the sense that, we can allow for node degree
level perturbations subject to the constraints posed in \cref{thm:CLX18}, and we still have
``weak recovery''. However, the fact that the recovery error depends on the perturbation levels
is not ideal, although we can impose further constraints on the perturbation levels to get a more
explicit robustness guarantee. This in part inspires our extension of the edge-robust weak recovery 
algorithm in \cref{sec:MS16} to the perturbed SBM in \cref{sec:perturb.robust}, where we allow
for individual edge probabilities to be perturbed by some amount. We show that as long as the
perturbation levels are not too large or too structured, we can still achieve weak recovery
and the result does not depend on the perturbation levels.

\subsection{Other robustness results}
In this section, we will briefly cover some other robustness results using an SDP approach in the literature.
\subsubsection{Edge-robust weak recovery at the KS threshold}
In \cref{sec:MS16}, we saw an algorithm that achieves weak recovery for all $d$ sufficiently large.
However, it remained unclear whether an algorithm can achieve weak recovery at the Kesten-Stigum 
threshold for all $d$. In 2022, Ding et al. \cite{DdNS22} built a much more complicated algorithm,
that finally achieves weak recovery at the Kesten-Stigum threshold for all $d$, and is robust 
against edge perturbations. 
\begin{theorem}[Edge-robust weak recovery at KS threshold, Theorem 5.3 of \cite{DdNS22}]
Let $G \sim \mathcal{G}(n, d, \epsilon)$ and assume that $\lambda^2 = \epsilon^2 d / 4 \geq 1 + \delta$ 
for some $\delta > 0$. Now suppose $G^\circ$ is obtained by modifying $G$ in at most $\rho n$ edges,
where
\[
\rho \leq \left(\frac{1}{\delta}\log\frac{1}{\epsilon}\right)^{-O(1/\delta)}.
\]
Then, there exists a polynomial-time algorithm that, with high probability, outputs $\hat{\bm{x}}$ 
satisfying
\[
\mathbb{E}\langle \bm{x}, \hat{\bm{x}} \rangle^2 \geq \|\bm{x}\|^2 \cdot \delta^{O(1)}.
\]
\end{theorem}
Their algorithm uses the idea of a constant length self-avoiding walk matrix that is a noisy but 
robust version of the label matrix $\bm{x}\bm{x}^\top$. It is defined as follows: for
each pair of vertices $i, j \in [n]$, we have
\[
\bm{Q}^{(s)} = \sum_{\gamma \in \mathrm{SAW}_s(i,j)} \prod_{(u,v) \in \gamma} 
(\bm{A}_G^\mathrm{cent,trunc})_{uv},
\]
where $\mathrm{SAW}_s(i,j)$ is the set of all length-$s$ self-avoiding walks from $i$ to $j$, and
$\bm{A}_G^\mathrm{cent,trunc}$ is the truncated centered adjacency matrix, where all the high degree
nodes are truncated. Then, they look at the SDP problem of maximizing $\langle \bm{Q}^{(s)^2}, \bm{X} \rangle$ 
subject to $\bm{X}$ being positive semidefinite plus other constraints, and finally do a rounding procedure
on the optimizer to get the estimator $\hat{\bm{x}}$.

\subsubsection{Weak recovery with outlier nodes}
Consider the setting where we have a graph with $n + m$ nodes, where $n$ of them are generated
from a balanced, two-community SBM, and the other $m$ nodes are outlier nodes that can be connected
to the rest of the graph in an arbitrary way. This is a stronger perturbation model in the sense that
we can potentially change $O(nm + m^2)$ edges.

It turns out that we can still achieve weak recovery in this setting. Cai and Li \cite{CL15} 
investigated the following SDP problem, that comes from a relaxation of the maximum likelihood estimator. 
Writing $\bm{J}$ for the all-ones matrix,
\begin{align*}
\text{minimize} \quad & \langle \bm{X}, -\bm{A}_G + \alpha \bm{I} + \lambda (\bm{J} - \bm{I}) \rangle\\
\text{subject to} \quad & \bm{X} \succeq 0, \\
& \bm{X}_{ij} \in [0,1], \forall i,j \in [n+m].
\end{align*}
Their main result is that the optimizer of this SDP has a nice block structure.
\begin{theorem}[SDP for SBM with outlier nodes, Theorem 3.1 of \cite{CL15}]
Let $G$ be a graph with $n + m$ nodes, where $n$ of them are generated from a balanced, two-community
SBM with parameters $p$ and $q$, and the other $m$ nodes are outlier nodes that can be connected to 
the rest of the graph in an arbitrary way. Assume that $p > C\log n / n$, $\alpha > 3m$, and that for
some $\delta$ satisfying
\[
\delta > C\left(\sqrt{\frac{\log n}{n}} + \frac{\alpha}{n} + \frac{q}{\sqrt{n}} + \frac{m}{n}
+ \frac{mp}{\alpha - 2m}\right),
\]
the tuning parameter $\lambda$ satisfies $q + \delta < \lambda < p - \delta$. Then, with probability
$1 - O(n^{-1})$, up to permutation of node orders, the optimizer $\hat{\bm{X}}$ of the above SDP has
the following block structure:
\[
\hat{\bm{X}} = \begin{bmatrix} 
  \bm{J}_{n/2} & \bm{0} & \bm{Z}_1\\
  \bm{0} & \bm{J}_{n/2} & \bm{Z}_2\\
  \bm{Z}_1^\top & \bm{Z}_2^\top & \bm{W}
\end{bmatrix},
\]
where $\bm{Z}_1$, $\bm{Z}_2$ and $\bm{W}$ are matrices of appropriate dimensions, and can be arbitrary.
\end{theorem}
As nice as this result is, in practice it is still hard to use it to recover the community labels 
of the inlier nodes. Cai and Li propose to use a $k$-means clustering algorithm on the normalized
columns of $\hat{\bm{X}}$ to get an estimator $\hat{\bm{x}}$. They were able to show the following
weak recovery guarantee.
\begin{theorem}[Recovery for SBM with outlier nodes, Theorem 3.2 of \cite{CL15}]
With all the assumptions in the previous theorem, if $m < n/8$, then, with high probability, 
the misclassification rate of $\hat{\bm{x}}$ on the $n$ inlier nodes is less than or equal to $7m/n$.
\end{theorem}

\subsubsection{Node-robust weak recovery}
The previous setting investigated the case where $p$ scales as $C\log n / n$, which is a much denser
regime than the one we have been considering in \cref{sec:MS16}, where $p$ scales as $d/n$ for some
constant $d$. In fact, we can also achieve weak recovery in the sparse regime with a slightly
different notion of node perturbations. Instead of having $m$ outlier nodes that can be connected to
the rest of the graph, we allow for $\mu n$ node edits on the original graph, where each node edit
can change the edges between the edited node and all the other nodes in an arbitrary way.

Ding et al. \cite{DdHS23} proposed a modification of the algorithm in \cref{sec:MS16} that can
achieve weak recovery in this setting. Note that we can potentially change $O(n^2)$ edges through 
node edits, exceeding the $O(n\sqrt{d})$ edge perturbation robustness of the original algorithm.

Recall that the main idea behind the original algorithm is to leverage the push-out effect of the
SDP value of the centered adjacency matrix, i.e., the fact that $\mathcal{SDP}(\bm{A}_G^{\mathrm{cent}}/\sqrt{d})$
and $\mathcal{SDP}(\bm{A}_G^{\mathrm{cent}}/\sqrt{d} - (\lambda / n)\bm{x}\bm{x}^\top)$ are bounded 
apart, which then guarantees that the optimizer of $\mathcal{SDP}(\bm{A}_G^{\mathrm{cent}}/\sqrt{d})$
must contain nontrivial information about $\bm{x}$. 

In the algorithm of Ding et al., the key observation is that the uncorrupted vertices still form a 
large principal submatrix of $\bm{A}_G^{\mathrm{cent}}$ on which the push-out effect holds. Since 
this clean set is unknown, they search for a large principal submatrix whose SDP value is large and
whose operator norm is bounded. The large SDP value provides the signal, while the operator norm bound
ensures that deleting any small subset of nodes cannot account for the objective value. Thus the large
SDP value must survive on the clean submatrix of this chosen submatrix, where we can then
utilize the push-out effect to get nontrivial information about $\bm{x}$.

\section{Proof of weak recovery \& edge robustness (Theorem \ref{thm:MS16.recovery})}
\label{sec:weak.recovery}
Before we start the proof, we will make a few definitions, and show a few preliminary results.
Let $\bm{x}$ be the label vector of $G \sim \mathcal{G}(n, d, \epsilon)$. Note that 
$G_\mathrm{sample} \sim \mathcal{G}(n, d_\mathrm{sample}, \epsilon)$, 
and $G_\mathrm{eval} \sim \mathcal{G}(n, d_\mathrm{eval}, \epsilon)$, 
where $d_\mathrm{sample} = d/(1 + n^{-1/2})$ and $d_\mathrm{eval} = n^{-1/2}d_\mathrm{sample}$.
For simplicity, from now on we write $d = d_\mathrm{sample}$.

Throughout the proof, we will use the constant $C$ to denote a positive constant that changes 
from line to line. We will often use the fact that, if two events both hold with probability 
at least $1 - Ce^{-n/C}$, then their intersection also holds with probability at least $1 - Ce^{-n/C}$, 
by a simple union bound, and
\[
C_1 e^{-n/C_1} + C_2 e^{-n/C_2} \leq (C_1 + C_2) e^{-n/\max(C_1, C_2)} 
\leq (C_1 + C_2) e^{-n/(C_1 + C_2)} = C e^{-n/C}.
\]

Also, note that, in our definition, $G_\mathrm{sample}$ and $G_\mathrm{eval}$ are dependent, since
they cannot share edges. However, we need their independence in our proof. To fix this issue, we note
that if we sampled them independently, then the two graphs would share $O(1)$ edges with high probability,
which we can ignore since we will show that the algorithm is robust against $o(n\sqrt{d})$ edge perturbations.
Hence, we can assume that $G_\mathrm{sample}$ and $G_\mathrm{eval}$ are independent without loss of generality.

\subsection{Step 1: $\bm{X}^*$ contains nontrivial information about $\bm{x}$}
We first introduce a group of Gaussian random matrices (GOEs) that have well-behaved SDP values.
\begin{definition}[Gaussian orthogonal ensemble (GOE) and the deformed GOE $\bm{B}(\lambda)$]
\label{def:GOE}
A random matrix $\bm{W} \in \mathbb{R}^{n \times n}$ is a \emph{GOE matrix} 
if it is a symmetric matrix with $\{\bm{W}_{ij}\}_{i \leq j}$ independent
and $\bm{W}_{ij} \sim \mathcal{N}(0, 1/n)$ for $i < j$ and $\bm{W}_{ii} \sim \mathcal{N}(0, 2/n)$. 
The \emph{deformed GOE matrix} $\bm{B}(\lambda)$ is then defined as \[\bm{B}(\lambda) = \frac{\lambda}{n} \bm{1}\bm{1}^\top + \bm{W}.\]
\end{definition}

We then have the following phase transition result for the SDP value of $\bm{B}(\lambda)$.
\begin{lemma}[SDP value of $\bm{B}(\lambda)$, Theorem 5 of \cite{MS16}]
\label{lem:GOE.SDP}
Let $\bm{B}(\lambda)$ be a deformed GOE matrix defined in \cref{def:GOE}. Then, with probability converging to 1 as $n \to \infty$,
\begin{itemize}
\item[(a)] if $\lambda \leq 1$: for any $\epsilon > 0$, we have 
$\mathcal{SDP}(\bm{B}(\lambda)) / n \in [2 - \epsilon, 2 + \epsilon]$.
\item[(b)] if $\lambda > 1$: there exists $\Delta(\lambda) > 0$ such that 
$\mathcal{SDP}(\bm{B}(\lambda)) / n \geq 2 + \Delta(\lambda)$.
\end{itemize}
\end{lemma}

If we set the diagonal entries of $\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}$ to be $\lambda \sqrt{d}/n$, then 
\[\frac{1}{\sqrt{d}}\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}} = \frac{\lambda}{n} \bm{x}\bm{x}^\top + \bm{E},\]
where $\bm{E}$ is a symmetric matrix with zero diagonal and independent off-diagonal entries satisfying
\[\bm{E}_{ij} = \begin{cases}
  \frac{1}{\sqrt{d}} - \frac{\sqrt{d}}{n} - \frac{\lambda}{n} \bm{x}_i\bm{x}_j & \text{with probability } \mathbb{P}((i,j) \in E_\mathrm{sample}), \\
  -\frac{\sqrt{d}}{n} - \frac{\lambda}{n} \bm{x}_i\bm{x}_j & \text{with probability } 1 - \mathbb{P}((i,j) \in E_\mathrm{sample}).
\end{cases}\]
Let $\bm{X}^*$ be an optimizer of $\mathcal{SDP}(\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}})$. We will now try to bound $\langle \bm{E}, \bm{X}^* \rangle$ and $\langle \frac{1}{\sqrt{d}}\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}, \bm{X}^* \rangle$
away from each other using the phase transition result of \cref{lem:GOE.SDP}, which will then give us 
a lower bound on $n^{-2} \langle \bm{x}\bm{x}^\top, \bm{X}^* \rangle$, 
which is the correlation between $\bm{X}^*$ and the ground truth labels matrix $\bm{x}\bm{x}^\top$.

The following lemma bounds $\mathcal{SDP}(\bm{E})$ to the SDP value of a GOE matrix.
\begin{lemma}[GOE approximation of $\mathcal{SDP}(\bm{E})$]
\label{lem:SDP.approx1}
Let $\bm{W}$ be a GOE matrix. Then, there exists $C = C(\lambda) > 0$ such that, for all $n \geq n_0 (d, \epsilon)$
with probability at least $1 - Ce^{-n/C}$, we have
\[
\left|\frac{1}{n} \mathcal{SDP}(\bm{E}) - \frac{1}{n} \mathcal{SDP}(\bm{W})\right| \leq \frac{C\log(d)}{d^{1/10}}.
\]
Moreover, $C(\lambda)$ is bounded for $\lambda$ in any compact set.
\end{lemma}

We also have a similar result bounding $\mathcal{SDP}(\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}/\sqrt{d})$
by the SDP value of a deformed GOE matrix.
\begin{lemma}[Deformed GOE approximation of $\mathcal{SDP}(\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}/\sqrt{d})$]
\label{lem:SDP.approx2}
Let $\bm{B}(\lambda)$ be a deformed GOE matrix. Then, there exists $C = C(\lambda) > 0$ such that, for all $n \geq n_0 (d, \epsilon)$
\[
\left|\frac{1}{n} \mathcal{SDP}\left(\frac{1}{\sqrt{d}}\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}\right) - \frac{1}{n} \mathcal{SDP}(\bm{B}(\lambda))\right| \leq \frac{C\log(d)}{d^{1/10}}.
\]
Moreover, $C(\lambda)$ is bounded for $\lambda$ in any compact set.
\end{lemma}

We will discuss the proofs of the above two lemmas later in \cref{sec:sdp.approx}. We will take them for granted here.

Now we have the tools to obtain a lower bound on $n^{-2} \langle \bm{x}\bm{x}^\top, \bm{X}^* \rangle$.
\begin{align*}
\left\langle \frac{\lambda}{n^2} \bm{x}\bm{x}^\top, \bm{X}^* \right\rangle & 
= \frac{1}{n}\left(\left\langle \frac{1}{\sqrt{d}}\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}, \bm{X}^* \right\rangle - \left\langle \bm{E}, \bm{X}^* \right\rangle\right)\\
& = \frac{1}{n}\mathcal{SDP}\left(\frac{1}{\sqrt{d}}\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}\right) - \frac{1}{n}\left\langle \bm{E}, \bm{X}^* \right\rangle\\
& \geq \frac{1}{n}\mathcal{SDP}\left(\frac{1}{\sqrt{d}}\bm{A}_{G_\mathrm{sample}}^{\mathrm{cent}}\right) - \frac{1}{n}\mathcal{SDP}(\bm{E})
&& \text{by optimality of SDP}\\
& \geq \frac{1}{n}\mathcal{SDP}(\bm{B}(\lambda)) - \frac{1}{n}\mathcal{SDP}(\bm{W}) - \frac{C\log(d)}{d^{1/10}}
&& \text{by Lemmas \ref{lem:SDP.approx1} and \ref{lem:SDP.approx2}}\\
& \geq \Delta_1(\lambda) - \frac{C\log(d)}{d^{1/10}},
&& \text{by Lemma \ref{lem:GOE.SDP}}
\end{align*}
with probability at least $1 - Cn^{-4}$ (see the remark below).
\begin{remark}
In the original paper \cite{MS16}, the authors claimed that the above inequality holds with probability at least $1 - Ce^{-n/C}$, 
but tracking the proof of Lemma \ref{lem:GOE.SDP} (Appendices F and G of \cite{MS16}) shows that 
they only proved the result with probability at least $1 - Cn^{-4}$ (Lemma G.8 of \cite{MS16}).
This is still sufficient for our purposes, as the probability still tends to 1 as $n \to \infty$.
\end{remark}
Now fix $\lambda$, and take $d$ sufficiently large such that $\Delta_1(\lambda) - \frac{C\log(d)}{d^{1/10}} \geq \lambda \Delta_2(\lambda) > 0$.
Then, we have, with probability at least $1 - Cn^{-4}$,
\[n^{-2}\langle \bm{x}\bm{x}^\top, \bm{X}^* \rangle \geq \Delta_2(\lambda).\]
This is an important milestone, as it shows that $\bm{X}^*$ has a nontrivial correlation with the ground truth $\bm{x}\bm{x}^\top$.

\subsection{Step 2: There exists ``good'' label estimators among $\{\hat{\bm{x}}^{(i,j)}\}_{i,j \in [n]}$}
We first define the following rounding function used in step \ref{alg:MS16.rounding} of \cref{alg:MS16}:
\begin{definition}[Rounding function]
For $t \geq 0$, define the \emph{rounding function} $r_t: \mathbb{R} \to \{-1, 0, +1\}$ as follows:
\[r_t(x) = 
\begin{cases}
1 & \text{if } x \geq t,\\
0 & \text{if } -t < x < t,\\
-1 & \text{if } x \leq -t.
\end{cases}
\]
\end{definition}
Hence, we can write $\hat{\bm{x}}^{(\hat{\imath}, j)} = r_{\bm{v}^{(\hat{\imath})}_j}(\bm{v}^{(\hat{\imath})})$.

Since $\bm{X} \in \mathcal{PSD}_1(n)$, writing its eigen-decomposition as $\bm{X} = \sum_{i=1}^{n} \xi_i \bm{v}^{(i)} (\bm{v}^{(i)})^\top$,
where $\xi_i$ is the $i$-th largest eigenvalue, and $\bm{v}^{(i)}$ is the corresponding unit length eigenvector, 
we have $\xi_i \geq 0$ for all $i$ by positive semidefiniteness, and $\sum_{i=1}^{n} \xi_i = \operatorname{Tr}(\bm{X}) = n$. Thus,
\[
\Delta_2 (\lambda) \leq \frac{1}{n^2} \langle \bm{x}\bm{x}^\top, \bm{X}^* \rangle 
= \frac{1}{n^2} \sum_{i=1}^{n} \xi_i (\bm{x}^\top \bm{v}^{(i)})^2
= \sum_{i=1}^{n} \frac{\xi_i}{n} \cdot \left(\frac{\bm{x}^\top \bm{v}^{(i)}}{\sqrt{n}}\right)^2.
\]
So there exists $\hat{\imath} \in [n]$ such that $|\bm{x}^\top \bm{v}^{(\hat{\imath})}| \geq \sqrt{n\Delta_2(\lambda)}$.

Next, we will use following lemma to justify that the rounding function
preserves this correlation, which shows that it is possible to extract a good label estimator.
\begin{lemma}[Rounding Lemma]
\label{lem:rounding}
Let $X$, $Y$ be random variables such that: $\mathbb{P}(X = +1) = \mathbb{P}(X = -1) = 1/2$, 
$\mathbb{E}(XY)\geq \epsilon > 0$, and $\mathbb{E}(Y^2) = 1$. Then, there exists $t^*$ depending 
on the joint distribution of $X$ and $Y$, such that
\[
\mathbb{E}(X r_{t^*}(Y)) \geq \frac{\epsilon^2}{2}.
\]
\end{lemma}
\begin{proof}
Note that for all $t$, we have $X r_t(Y) = r_t(XY)$, and $Y^2 = (XY)^2$. Let $Z = XY$, then it suffices to show
that given $\mathbb{E}(Z) \geq \epsilon > 0$ and $\mathbb{E}(Z^2) = 1$, we have $\mathbb{E}(r_t(Z)) \geq \epsilon^2/2$.
Recall that for nonnegative random variable $S$, by Fubini, we have
\[
\mathbb{E}(S) = \mathbb{E}\left(\int_{0}^{\infty}\mathbbm{1}_{s\leq S}ds\right) = \int_{0}^{\infty}\mathbb{P}(S\geq s)ds,\quad
\mathbb{E}(S^2) = \mathbb{E}\left(\int_{0}^{\infty}2s\mathbbm{1}_{s\leq S}ds\right) = \int_{0}^{\infty}2s\mathbb{P}(S\geq s)ds.
\]
Therefore, writing $Z^\pm = \max(\pm Z, 0)$, we have
\begin{align*}
\epsilon & \leq \mathbb{E} (Z) = \mathbb{E} (Z_+) - \mathbb{E} (Z_-) = \int_{0}^{\infty} \mathbb{P}(Z_+\geq z) dz - \int_{0}^{\infty} \mathbb{P}(Z_-\geq z) dz\\
& \leq \int_{0}^{K} (\mathbb{P}(Z\geq z) - \mathbb{P}(Z\leq -z))dz + \int_{K}^{\infty} \frac{2z}{2K} \mathbb{P}(Z\geq z) dz - \int_{K}^{\infty} \frac{2z}{2K} \mathbb{P}(-Z\geq z) dz\\
& = \int_{0}^{K} \mathbb{E}{r_t(Z)}dt + \frac{1}{2K}\mathbb{E}(Z_+^2 - Z_-^2) = \int_{0}^{K} \mathbb{E}{r_t(Z)}dt + \frac{1}{2K}.
\end{align*}
Now take $K = 1/\epsilon$, then
\[
\frac{1}{K}\int_{0}^{K} \mathbb{E}{r_t(Z)}dt \geq \frac{1}{K}(\epsilon - \frac{1}{2K}) = \frac{\epsilon^2}{2}
\]
So the average of $\mathbb{E}{r_t(Z)}$ over $t\in [0, 1/\epsilon]$ is greater than $\epsilon^2/2$,
so there exists $t^* \in [0, 1/\epsilon]$ such that $\mathbb{E}{r_{t^*}(Z)} \geq \epsilon^2/2$.
\end{proof}

Without loss of generality, assume that $\bm{x}^\top \bm{v}^{(\hat{\imath})} > 0$. Consider $(X, Y)$ drawn uniformly from \\
$\{(\bm{x}_k, \sqrt{n}\bm{v}^{(\hat{\imath})}_k)\}_{k \in [n]}$, then
we have $\mathbb{E}(XY) = n^{-1/2} \bm{x}^\top \bm{v}^{(\hat{\imath})} \geq \sqrt{\Delta_2(\lambda)}$
and $\mathbb{E}(Y^2) = n^{-1} || \sqrt{n}\bm{v}^{(\hat{\imath})} ||^2 = || \bm{v}^{(\hat{\imath})} ||^2 = 1.$
Thus, by \cref{lem:rounding}, there exists $t^* \in [0, 1/\sqrt{\Delta_2(\lambda)}]$ such that
\[
\frac{\Delta_2(\lambda)}{2} \leq \mathbb{E}(X r_{t^*}(Y))
= \frac{1}{n} \sum_{k=1}^{n} \bm{x}_k r_{t^*}(\sqrt{n}\bm{v}^{(\hat{\imath})}_k)
= \frac{1}{n} \langle \bm{x}, r_{\tilde{t}}(\bm{v}^{(\hat{\imath})}) \rangle,
\]
where $\tilde{t} = t^*/\sqrt{n} \in [0, 1/\sqrt{n\Delta_2(\lambda)}]$. 
\begin{remark}
Here, we slightly improved the rounding lemma bound from $\epsilon^2/4$ in \cite{MS16} to $\epsilon^2/2$,
although the main proof idea remains the same. Also, the original application of the rounding lemma
missed the scaling factor $\sqrt{n}$ in the definition of $Y$, but the conclusion remains valid.
\end{remark}
Finally, note that for any value of $\tilde{t}$, we can find $\hat{\jmath} \in [n]$ such that 
$r_{\tilde{t}}(\bm{v}^{(\hat{\imath})}) = r_{\bm{v}^{(\hat{\imath})}_{\hat{\jmath}}}(\bm{v}^{(\hat{\imath})}) = \hat{\bm{x}}^{(\hat{\imath}, \hat{\jmath})}$.
Therefore, there exists some $\hat{\imath}, \hat{\jmath} \in [n]$ such that
\begin{equation}
\label{eqn:correlation}
\frac{1}{n} \langle \bm{x}, \hat{\bm{x}}^{(\hat{\imath}, \hat{\jmath})} \rangle \geq \frac{\Delta_2(\lambda)}{2}.
\end{equation}

Now, it remains to show that the selection step (Step \ref{alg:MS16.selection}) of \cref{alg:MS16}
will find such $\hat{\imath}, \hat{\jmath}$ with high probability, which will then complete the proof of Theorem \ref{thm:MS16.recovery}.

\subsection{Step 3: A ``good'' estimator can be found with $G_\mathrm{eval}$}
We first want to relate the quantities $\hat{\bm{x}}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}$,
the objective function in Step \ref{alg:MS16.selection} of \cref{alg:MS16},
to $\langle\bm{x}, \hat{\bm{x}}\rangle$, the correlation above, with high probability. 
Recall the well-known Azuma-Hoeffding inequality (see, for example, Alon and Spencer~\cite[Chapter 7]{Alon16}).
\begin{lemma}[Azuma-Hoeffding inequality]
\label{lem:A-H}
Let $X_0, \ldots, X_m$ be a martingale, with bounded increments,
i.e. $|X_i - X_{i-1}| \leq c_i$ for all $i \in [m]$. Then, for all $t > 0$, we have
\[
\mathbb{P}(|X_m - \mathbb{E}(X_m)| \geq t) \leq 2\exp\left(-\frac{2t^2}{\sum_{i=1}^{m} c_i^2}\right).
\]
\end{lemma}
We apply this inequality to the objective function in Step \ref{alg:MS16.selection} of \cref{alg:MS16}, 
and obtain the following concentration result.
\begin{lemma}
\label{lem:conc}
Let $G \sim \mathcal{G}(n, d, \epsilon)$. For any fixed vector $\hat{\bm{x}} \in \{-1, 0, +1\}^n$, 
define $f(G; \hat{\bm{x}}) = \hat{\bm{x}}^\top \bm{A}_{G}^{\mathrm{cent}} \hat{\bm{x}}.$
Then, for any $t > 0$,
\[
\mathbb{P}\Bigl(
    \bigl|f(G; \hat{\bm{x}})-\mathbb{E}(f(G; \hat{\bm{x}}))\bigr|
    \ge t
    \;\Bigm|\;
    |E_{\mathrm{eval}}| \le m
\Bigr) \leq 2 \exp\left(-\frac{t^2}{8m}\right).
\]
\end{lemma}
\begin{proof}
Note that $f(G; \hat{\bm{x}}) = \hat{\bm{x}}^\top \bm{A}_{G}^{\mathrm{cent}} \hat{\bm{x}}
= \hat{\bm{x}}^\top \bm{A}_{G} \hat{\bm{x}} - (d/n) (\hat{\bm{x}}^\top \bm{1})^2$,
so writing $\tilde{f}(G; \hat{\bm{x}}) = \hat{\bm{x}}^\top \bm{A}_{G} \hat{\bm{x}}$,
we have $f(G; \hat{\bm{x}}) - \mathbb{E}(f(G; \hat{\bm{x}})) = \tilde{f}(G; \hat{\bm{x}}) - \mathbb{E}(\tilde{f}(G; \hat{\bm{x}}))$.
Note that $\tilde{f}(G; \hat{\bm{x}}) = \sum_{(a,b) \in E} 2\hat{\bm{x}}_a \hat{\bm{x}}_b$.

For some ordering of all possible edges, we can define a filtration $\mathcal{F}_0, \mathcal{F}_1, \ldots, \mathcal{F}_m$
with $\mathcal{F}_k = \sigma(E_1, \ldots, E_k)$, where $E_k \in E(G) \cup \varnothing$ is the $k$-th edge of $G$ in the ordering. 
Then, define the martingale $X_k = \mathbb{E}(\tilde{f}(G; \hat{\bm{x}}) | \mathcal{F}_k)$ for $k \in [m]$. 
Since $|E(G)| \leq m$, we have $X_m = \tilde{f}(G; \hat{\bm{x}})$. Note that for all $k \in [m]$, we have
\begin{align*}
|X_k - X_{k-1}| &= \left|\mathbb{E}(\tilde{f}(G; \hat{\bm{x}}) | E_{1:k}) - \mathbb{E}(\tilde{f}(G; \hat{\bm{x}}) | E_{1:k-1})\right|\\
& \leq \sup_{e_{1:k-1}, e_k, e_k'} \left|\mathbb{E}(\tilde{f}(G; \hat{\bm{x}}) | E_{1:k-1} = e_{1:k-1}, E_k = e_k) 
- \mathbb{E}(\tilde{f}(G; \hat{\bm{x}}) | E_{1:k-1} = e_{1:k-1}, E_k = e_k')\right|\\
& \leq \sup_{E(G^+) = E(G) \cup \{(a,b)\}} \left|\tilde{f}(G^+; \hat{\bm{x}}) - \tilde{f}(G; \hat{\bm{x}})\right|\\
& \leq \sup_{a,b} |2\hat{\bm{x}}_a \hat{\bm{x}}_b| \leq 2.
\end{align*}
Hence, we can apply the Azuma-Hoeffding inequality \cref{lem:A-H} to $X$ with $c_i = 2$ for all $i \in [m]$ to obtain
\begin{align*}
\mathbb{P}\Bigl(
    \bigl|f(G; \hat{\bm{x}})-\mathbb{E}(f(G; \hat{\bm{x}}))\bigr|
    \ge t
    \;\Bigm|\;
    |E(G)| \le m
\Bigr)
&= \mathbb{P}\Bigl(
    \bigl|\tilde{f}(G; \hat{\bm{x}})-\mathbb{E}(\tilde{f}(G; \hat{\bm{x}}))\bigr|
    \ge t
    \;\Bigm|\;
    |E(G)| \le m
\Bigr)\\
&= \mathbb{P}(|X_m - \mathbb{E}(X_m)| \geq t)\\
&\leq 2\exp\left(-\frac{t^2}{8m}\right).
\end{align*}
as desired.
\end{proof}
To apply the above concentration result, we need to bound 
$|E_\mathrm{eval}| \sim \operatorname{Binomial}\left(|E|, n^{-1/2}/(1 + n^{-1/2})\right)$,
which we can achieve with the following Chernoff bound argument.
\begin{lemma}[Bounding $|E_\mathrm{eval}|$]
\label{lem:eval.size}
Let $p_n = n^{-1/2}/(1 + n^{-1/2})$. Then, for some constant $C > 0$, we have
\[
\mathbb{P}(|E_\mathrm{eval}| \geq C \sqrt{n}) \leq Ce^{-\sqrt{n}/C}.
\]
\end{lemma}
\begin{proof}
Recall that $|E| \sim \operatorname{Binomial}\left(\binom{n}{2}, d/n\right)$, so by Chernoff bound, for all $s > 0$, we have
\begin{align*}
  \mathbb{P}(|E| \geq K_0 n) & \leq e^{-s K_0 n} \mathbb{E}(e^{s|E|}) 
  = e^{-sK_0 n} \left(1 - \frac{d}{n} + \frac{d}{n} e^{s}\right)^{\binom{n}{2}}\\
  & \leq \exp\left(-s K_0 n + \frac{d}{n} (e^{s} - 1) \binom{n}{2}\right)\\
  & \leq \exp\left(\left(\frac{d}{2} (e^{s} - 1) - sK_0\right) n\right)\\
  & \leq C_0e^{-n/C_0},
\end{align*}
where the last inequality holds when $K_0$ is sufficiently large, and $C_0$ is a constant depending on $d$.

Now, on the event $\{|E| \leq K_0 n\}$ with probability at least $1 - C_0e^{-n/C_0}$, 
we have $|E_\mathrm{eval}| \sim \operatorname{Binomial}\left(|E|, p_n\right)$,
so by Chernoff bound again, for all $s > 0$, we have
\begin{align*}
  \mathbb{P}(|E_\mathrm{eval}| \geq K_1 \sqrt{n})
  & = \mathbb{E} \left(\mathbb{P}(|E_\mathrm{eval}| \geq K_1 \sqrt{n} \mid |E|)\right)
  \leq e^{-sK_1\sqrt{n}} \mathbb{E}(\mathbb{E}(e^{s|E_\mathrm{eval}|} \mid |E|))\\
  & = e^{-sK_1\sqrt{n}} \mathbb{E}(\left(1 - p_n + p_n e^s\right)^{|E|})
  \leq e^{-sK_1\sqrt{n}} \left(1 + p_n (e^s - 1)\right)^{K_0 n}\\
  & = \exp\left(-sK_1\sqrt{n} + K_0 (e^s - 1) n p_n \right)
  \leq \exp\left((K_0 (e^s - 1) - sK_1) \sqrt{n}\right)\\
  & \leq C_1e^{-\sqrt{n}/C_1},
\end{align*}
where the last inequality holds when $K_1$ is sufficiently large, and $C_1$ is a constant depending on $d$.
Combining the two probability bounds, we have
\[
\mathbb{P}(|E_\mathrm{eval}| \geq K \sqrt{n}) \leq C_0e^{-n/C_0} + C_1e^{-\sqrt{n}/C_1} \leq Ce^{-\sqrt{n}/C},
\]
for suitable constants $C$, as desired.
\end{proof}
By Lemmas \ref{lem:conc} and \ref{lem:eval.size}, for all $s \in [0,1]$, we have,
with probability at least $1 - Ce^{-\sqrt{n}s^2/C}$,
\[
\left|\hat{\bm{x}}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}} 
- \mathbb{E}\left(\hat{\bm{x}}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}\right)\right| \leq s \sqrt{n}.
\]
Note that
\begin{align*}
\mathbb{E}\left(\hat{\bm{x}}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}\right)
&= \hat{\bm{x}}^\top \mathbb{E}\left(\bm{A}_{G_\mathrm{eval}}^\mathrm{cent}\right) \hat{\bm{x}}\\
&= \hat{\bm{x}}^\top \frac{\epsilon d_\mathrm{eval}}{2n} \left(\bm{x}\bm{x}^\top - \bm{I}_n\right) \hat{\bm{x}}\\
&= \frac{\epsilon d_\mathrm{eval}}{2n} (\hat{\bm{x}}^\top \bm{x})^2 - \frac{\epsilon d_\mathrm{eval}}{2}\\
&= \frac{\delta_n \epsilon d}{2} \left(\frac{1}{n} \langle \bm{x}, \hat{\bm{x}} \rangle^2 - 1\right),
\end{align*}
where $d_\mathrm{eval} = \delta_n d$ is the average degree of $G_\mathrm{eval}$, and
$\delta_n = n^{-1/2}/(1 + n^{-1/2}) = O(n^{-1/2})$ is the fraction of edges in $G_\mathrm{eval}$ among all edges in $G$.
Thus, with probability at least $1 - Ce^{-\sqrt{n}s^2/C}$, we have
\begin{equation}
\label{eqn:conc}
  \left|\hat{\bm{x}}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}} 
  - \frac{\delta_n \epsilon d}{2n} \langle\bm{x}, \hat{\bm{x}}\rangle^2 + \frac{\delta_n \epsilon d}{2}\right| \leq s \sqrt{n}.
\end{equation}
This concentration result relates $\hat{\bm{x}}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}$
to the correlation $\langle\bm{x}, \hat{\bm{x}}\rangle$ with high probability.

In particular, at the end of the previous step (\cref{eqn:correlation}), we have shown that 
there exists some $\hat{\imath}, \hat{\jmath} \in [n]$ such that 
$\frac{1}{n} \langle \bm{x}, \hat{\bm{x}}^{(\hat{\imath}, \hat{\jmath})} \rangle \geq \frac{\Delta_2(\lambda)}{2}$.
Then, by the concentration result \ref{eqn:conc}, with probability at least $1 - Ce^{-\sqrt{n}s^2/C}$, we have
\begin{align*}
  \frac{1}{n \delta_n} \hat{\bm{x}}^{(\hat{\imath}, \hat{\jmath})}{}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}^{(\hat{\imath}, \hat{\jmath})} 
  &\geq \frac{\epsilon d}{2} \left(\frac{1}{n} \langle \bm{x}, \hat{\bm{x}}^{(\hat{\imath}, \hat{\jmath})} \rangle\right)^2
  - \frac{\epsilon d}{2n} - \frac{s}{\sqrt{n}\delta_n}\\
  &\geq \frac{\epsilon d}{8} \Delta_2(\lambda)^2 - \frac{\epsilon d}{2n} - s\left(1 + \frac{1}{\sqrt{n}}\right)\\
  &\geq \frac{\epsilon d}{12} \Delta_2(\lambda)^2 =: \Delta_3 (\lambda),
\end{align*}
where the last inequality follows from 
\begin{equation}
\label{eqn:s.choice}
\frac{\epsilon d}{2n} + s\left(1 + \frac{1}{\sqrt{n}}\right) \leq \frac{\epsilon d}{24} \Delta_2(\lambda)^2,
\end{equation}
which holds for sufficiently large $n$ and sufficiently small $s$. Therefore,
\[
\max_{i, j \in [n]} \frac{1}{n\delta_n} \hat{\bm{x}}^{(i,j)}{}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}^{(i,j)}
\geq \frac{1}{n\delta_n} \hat{\bm{x}}^{(\hat{\imath}, \hat{\jmath})}{}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}^{(\hat{\imath}, \hat{\jmath})} 
\geq \Delta_3 (\lambda).
\]

On the other hand, suppose that $\hat{\bm{x}}$ has a lower correlation with $\bm{x}$, say
$\frac{1}{n} \langle \bm{x}, \hat{\bm{x}} \rangle \leq \frac{\Delta_2(\lambda)}{4}$, 
then by the concentration result \ref{eqn:conc}, with probability at least $1 - Ce^{-\sqrt{n}s^2/C}$, we have
\begin{align*}
\frac{1}{n\delta_n} \hat{\bm{x}}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}
&\leq \frac{\epsilon d}{2} \left(\frac{1}{n} \langle \bm{x}, \hat{\bm{x}} \rangle\right)^2 
+ \frac{\epsilon d}{2n} + s\left(1 + \frac{1}{\sqrt{n}}\right)\\
&\leq \frac{\epsilon d}{32} \Delta_2(\lambda)^2 + \frac{\epsilon d}{24} \Delta_2(\lambda)^2\\
&< \frac{\epsilon d}{12} \Delta_2(\lambda)^2 = \Delta_3 (\lambda).
\end{align*}

Thus, with probability at least $1 - Ce^{-\sqrt{n}/C}$, the maximizer 
$\hat{\bm{x}}^\mathrm{SDP} = \hat{\bm{x}}^{(i^*, j^*)}$ of 
$\max_{i, j \in [n]} \hat{\bm{x}}^{(i,j)}{}^\top \bm{A}_{G_\mathrm{eval}}^\mathrm{cent} \hat{\bm{x}}^{(i,j)}$
must satisfy
\[
\frac{1}{n} \langle \bm{x}, \hat{\bm{x}}^\mathrm{SDP} \rangle \geq \frac{\Delta_2(\lambda)}{4}.
\]
This completes the proof of the weak recovery guarantee in Theorem \ref{thm:MS16.recovery}.

\begin{remark}
In the original proof of \cite{MS16}, the authors missed the $\delta_n$ scaling factor. This leads
to them having $s/\sqrt{n}$ instead of $s(1 + 1/\sqrt{n})$ in \cref{eqn:s.choice}. Consequently, 
this inequality only has constant margin instead of an $O(\sqrt{n})$ margin suggested by the original
proof. Luckily, this does not affect the final conclusion.
\end{remark}

\subsection{Robustness against modifying $O(n\sqrt{d})$ edges}
Now, we turn to proving the robustness of the algorithm against adversarial edge modifications.
Which steps are affected by modifying $G$? Those are just the places we explicitly used
properties of $\bm{A}_{G}^\mathrm{cent}$, and they are Lemmas \ref{lem:SDP.approx1} and \ref{lem:SDP.approx2},
and Equation \ref{eqn:conc} in the selection step.

For the two lemmas, we make use of the ``bounded differences'' property of the SDP again.
For any two graphs $G, G'$ that differ by one edge $(u,v)$, let $\bm{X}^*$ and $\bm{X}'^*$ be the optimal solutions 
to $\mathcal{SDP}(\bm{A}_{G}^\mathrm{cent})$ and $\mathcal{SDP}(\bm{A}_{G'}^\mathrm{cent})$ respectively.
Then, we have
\begin{align*}
\mathcal{SDP}(\bm{A}_{G}^\mathrm{cent}) - \mathcal{SDP}(\bm{A}_{G'}^\mathrm{cent})
& = \langle \bm{A}_{G}^\mathrm{cent}, \bm{X}^* \rangle - \langle \bm{A}_{G'}^\mathrm{cent}, \bm{X}'^* \rangle\\
& \leq \langle \bm{A}_{G}^\mathrm{cent}, \bm{X}^* \rangle - \langle \bm{A}_{G'}^\mathrm{cent}, \bm{X}^* \rangle\\
& \leq |X_{uv}^*| + |X_{vu}^*| \leq 2,\\
\end{align*}
and similarly in the other direction. Hence, the SDP value can change by at most $2$ when we modify
one edge, so when we modify $o(n\sqrt{d})$ edges, the SDP value can change by at most $o(n\sqrt{d})$,
which is negligible compared to the SDP value of order $\Theta(n\sqrt{d})$ in the regime $\lambda > 1$.

For the selection step, note that for any $\hat{\bm{x}} \in \{-1, 0, +1\}^n$, we have
\[
|\hat{\bm{x}}^\top \bm{A}_{G}^\mathrm{cent} \hat{\bm{x}} - \hat{\bm{x}}^\top \bm{A}_{G'}^\mathrm{cent} \hat{\bm{x}}|
= |\hat{\bm{x}}^\top (\bm{A}_{G}^\mathrm{cent} - \bm{A}_{G'}^\mathrm{cent}) \hat{\bm{x}}|
\leq 2 |\hat{\bm{x}}_u \hat{\bm{x}}_v| = 2.
\]
Furthermore, since $G_\mathrm{eval}$ takes only $O(1/\sqrt{n})$ of the edges of $G$, when we modify
$o(n\sqrt{d})$ edges, with high probability, we will modify $o(\sqrt{nd})$ edges in $G_\mathrm{eval}$, 
which will change the $\hat{\bm{x}}^\top \bm{A}_{G}^\mathrm{cent} \hat{\bm{x}}$ term by at most
$o(\sqrt{nd})$. But the right hand side of the concentration result \ref{eqn:conc} is $s\sqrt{n}$,
so this change can be absorbed by the concentration result.

Therefore, having shown that these two steps remain valid under $o(n\sqrt{d})$ edge modifications, 
we can conclude that the algorithm is robust against these modifications, which completes the proof 
of Theorem \ref{thm:MS16.recovery}. \hfill $\blacksquare$

\section{Robustness against perturbed SBMs}
\label{sec:perturb.robust}
In this section, we look at an original notion of graph perturbation, where instead of directly 
modifying edges, we are allowed to perturb the edge probabilities by some random $h_{ij}$, 
and we will show that the same algorithm can still achieve weak recovery.
\subsection{Perturbed SBMs}
We first define the following more general class of random graph models, the \emph{perturbed SBM}. 
We will then show that \cref{alg:MS16} still works for this more general class of random graph
models.
\begin{definition}[Perturbed SBM]
A graph $G \sim \mathcal{G}(n, d, \epsilon)$ is a balanced, two-community $h$-perturbed stochastic block model graph
with $V = [n]$, uniformly random label vector $\bm{x} \in \{\pm 1\}^n$ satisfying $\bm{1}^\top \bm{x} = 0$,
and perturbation variables $h_{ij}$ satisfying $\mathbb{E}[h_{ij}] = 0$ for all $i, j \in [n]$,
if for each pair of vertices $i, j \in [n]$, we have
\[p_{ij} := \mathbb{P}((i,j) \in E) = \begin{cases}
\frac{d}{n}(1+\frac{\epsilon}{2}+h_{ij}) & \text{if } x_i = x_j, \\
\frac{d}{n}(1-\frac{\epsilon}{2}+h_{ij}) & \text{if } x_i \neq x_j.
\end{cases}\]
We write $\eta_{ij} = x_ix_j\epsilon/2 + h_{ij}$ for short.
\end{definition}

We have the following weak recovery result for the perturbed SBM.
\begin{theorem}[Weak recovery for perturbed SBM]
\label{thm:perturbed.SBM}
Let $G \sim \mathcal{G}(n, d, \epsilon)$ be a balanced, two-community $h$-perturbed SBM graph,
and let $\lambda = \epsilon \sqrt{d} / 2$. Suppose there exists $\alpha > 0$, such that:
\begin{enumerate}
  \item[(a)] the perturbation $h$ satisfies that $\|h\|_\infty \leq C d^{-2\alpha}$,
  \item[(b)] let $\bm{H}$ be the matrix with entries $H_{ij} = h_{ij}$, 
  then $\mathbb{E}(\|\bm{H}\|_\text{op}) \leq Cnd^{-1/2 - \alpha}$.
\end{enumerate}
If $\lambda > 1$, then there exists some constant $\Delta_4(\lambda) > 0$, such that with probability
at least $1 - Cn^{-4}$, the estimator $\hat{\bm{x}}^\mathrm{SDP}$ returned by \cref{alg:MS16} satisfies
\[
\frac{1}{n} \langle \bm{x}, \hat{\bm{x}}^\mathrm{SDP} \rangle \geq \Delta_4(\lambda).
\]
\end{theorem}
\begin{remark}
We can view the theorem above as a different notion of robustness of the algorithm, where instead of
directly editing edges, we are allowed to perturb the edge probabilities by some random $h_{ij}$.
\end{remark}

\subsection{Example perturbations}
In this section, to understand the conditions in \cref{thm:perturbed.SBM}, we will discuss some
example perturbations $h_{ij}$'s that satisfy these conditions. We will see that when the perturbation
is sufficiently unstructured, we can have more than $O(n\sqrt{d})$ edge modifications in expectation
and still satisfy the conditions, while when the perturbation is highly structured, the number
of edge modifications is much smaller in order to satisfy the conditions.

\subsubsection{Example 1: i.i.d. perturbations}
Suppose $h_{ij}$'s are i.i.d. uniform in $\{-Cd^{-2\alpha}, Cd^{-2\alpha}\}$ for some constant $C$.
Then, condition (a) is satisfied by definition, and condition (b) follows from the well-known result
on the operator norm of random matrices with i.i.d. Rademacher entries (for example, \cite[Theorem 4.4.3]{Ver18}),
which gives $\mathbb{E}(\|\bm{H}\|_\text{op}) \lesssim O(\sqrt{n}d^{-2\alpha})$, which is smaller than
$O(nd^{-1/2 - \alpha})$ for large $n$. Hence, under this perturbation, the algorithm can still 
achieve weak recovery.

Note that the expected number of edge modifications under this perturbation is
\[\frac{d}{n}\sum_{i < j} \mathbb{E}|h_{ij}| = O\left(nd^{1 - 2\alpha}\right),\]
which can be larger than $O(n\sqrt{d})$ when $\alpha < 1/4$. This shows that the algorithm can in fact
be robust against more than $O(n\sqrt{d})$ edge modifications in expectation, as long as the
perturbation is sufficiently unstructured.

\subsubsection{Example 2: structured perturbations}
Now consider the opposite case where $h_{ij}$'s are highly structured, say 
$h_{ij} = L\sigma \mathbbm{1}_{\{i\leq k, j\leq k\}}$ for some $k \in [n]$, where $L$ is a scaling
constant and $\sigma$ is a random variable uniform in $\{-1, +1\}$. Then, condition (a) is satisfied
if $L \leq Cd^{-2\alpha}$. However, note that regardless of $\sigma$, $\|\bm{H}\|_\text{op} = Lk$, 
so condition (b) requires $Lk \leq Cnd^{-1/2 - \alpha}$, which implies $L \leq C k^{-1} n d^{-1/2 - \alpha}$.
In this case, the expected number of edge modifications is
\[\frac{d}{n}\sum_{i < j} \mathbb{E}|h_{ij}| = O\left(kd^{1/2 - \alpha}\right),\]
usually much smaller than $O(n\sqrt{d})$ when $k \ll n$. This shows that when the perturbation is highly
structured, the algorithm can only be robust against much fewer edge modifications in expectation.

\section{Proof of weak recovery for perturbed SBMs (\cref{thm:perturbed.SBM})}
\label{sec:sdp.approx}
Note that the proof of weak recovery in \cref{sec:weak.recovery} only uses the adjacency matrix of the graph 
in two places: the SDP approximation lemmas (Lemmas \ref{lem:SDP.approx1} and \ref{lem:SDP.approx2}),
which we will discuss for the rest of this section,
and $\mathbb{E}[\bm{A}_{G_\mathrm{eval}}^\mathrm{cent}]$ in \cref{eqn:conc}, which will be the same
for the perturbed SBM and the original SBM since $\mathbb{E}[h_{ij}] = 0$ for all $i, j \in [n]$.
Thus, to show that the algorithm remains effective for the perturbed SBM,
we only need to show that similar SDP approximation results hold for the perturbed SBM.

Therefore, in this section, we will demonstrate how to get a similar SDP approximation 
result as in Lemmas \ref{lem:SDP.approx1} and \ref{lem:SDP.approx2}, which will prove theorem 
\ref{thm:perturbed.SBM}. The proofs of the lemmas will follow the exact same route, so they are
ommitted here for brevity. We refer the reader to the original paper \cite{MS16} for more details.

\subsection{Helper functions}
\label{sec:sdp.helper}
The main idea of approximating the SDP of adjacency matrices with the SDP of deformed GOE matrices
is to utilize the following two helper functions:
\begin{definition}[$\mathcal{OPT}_k$]
Define the rank-constrained SDP value for $\bm{M} \in \mathbb{R}^{n\times n}$ and $k \in [n]$:
\[
\mathcal{OPT}_k(\bm{M}) = \max_{\substack{\bm{X} \in \mathcal{PSD}_1(n)\\
\operatorname{rank}(\bm{X}) \leq k}} \langle \bm{M}, \bm{X} \rangle.
\]
Equivalently, let $\mathbb{S}^{k-1} = \{\bm{v} \in \mathbb{R}^k: ||\bm{v}||_2 = 1\}$ be the unit sphere
in $\mathbb{R}^k$, then we have
\[
\mathcal{OPT}_k(\bm{M}) = \max_{\bm{\sigma}_1, \dots, \bm{\sigma}_n \in \mathbb{S}^{k-1}}
\sum_{i,j=1}^n M_{ij} \langle \bm{\sigma}_i, \bm{\sigma}_j \rangle.
\]
\end{definition}
\begin{definition}[Smoothed OPT]
For $\beta > 0$, define $\Phi(\beta, k; \bm{M})$, the smoothed version of $\mathcal{OPT}_k(\bm{M})$,
as follows:
\[
\Phi(\beta, k; \bm{M}) = \frac{1}{\beta} \log \left(\int_{(\mathbb{S}^{k-1})^n} \exp\left(\beta \sum_{i,j=1}^n M_{ij}
\langle \bm{\sigma}_i, \bm{\sigma}_j \rangle\right) d\nu(\bm{\sigma})\right),
\]
where $\nu$ is the uniform probability measure on $(\mathbb{S}^{k-1})^n$.

We can see that as $\beta \to \infty$, $\Phi(\beta, k; \bm{M})$ will select the maximum value of
$\sum_{i,j=1}^n M_{ij} \langle \bm{\sigma}_i, \bm{\sigma}_j \rangle$, which is exactly $\mathcal{OPT}_k(\bm{M})$.
\end{definition}

In their paper, Montanari and Sen showed the following bounds relating $\mathcal{SDP}(\bm{M})$, 
$\mathcal{OPT}_k(\bm{M})$ and $\Phi(\beta, k; \bm{M})$. We will quote these results without proof.
\begin{theorem}[SDP vs OPT, Theorem 4 Equation 15 of \cite{MS16}]
\label{thm:SDP.OPT}
For $k \geq k_0$, where $k_0$ is a sufficiently large constant, for any symmetric matrix 
$\bm{M} \in \mathbb{R}^{n \times n}$, we have
\[
\left(1-\frac{1}{k}\right) \mathcal{SDP}(\bm{M}) - \frac{1}{k} |\mathcal{SDP}(-\bm{M})|
\leq \mathcal{OPT}_k(\bm{M}) \leq \mathcal{SDP}(\bm{M}).
\] 
\end{theorem}
\begin{remark}
Using just the first inequality, we have
\[
\mathcal{OPT}_k (\bm{M}) + \mathcal{OPT}_k (-\bm{M}) \geq \left(1 - \frac{2}{k}\right) 
\left(|\mathcal{SDP}(\bm{M})| + |\mathcal{SDP}(-\bm{M})|\right),
\]
so subtract by $\mathcal{SDP}(\bm{M})$ on every term and flipping signs, we get
\[
0 \leq \mathcal{SDP}(\bm{M}) - \mathcal{OPT}_k(\bm{M}) 
\leq \frac{1}{k} \left(\left|\mathcal{SDP}(\bm{M})\right| + \left|\mathcal{SDP}(-\bm{M})\right|\right)
\leq \frac{1}{k-2} \left(\mathcal{OPT}_k(\bm{M}) + \mathcal{OPT}_k(-\bm{M})\right).
\]
\end{remark}
\begin{remark}
Clearly, $\bm{A}^\mathrm{cen}_{G}$ and $\bm{B}(\lambda)$ are symmetric matrices. 
Montanari and Sen showed that with probability at least $1 - Ce^{-n/C}$, we have
(Theorem 8 equation 47 of \cite{MS16})
\[
\max \left\{\frac{1}{n\sqrt{d}}\mathcal{OPT}_k(\pm \bm{A}^\mathrm{cen}_{G}), 
\frac{1}{n}\mathcal{OPT}_k(\pm \bm{B}(\lambda))\right\} \leq C,
\]
so in fact, for $\bm{M} = \bm{A}^\mathrm{cen}_{G}$ or $\bm{B}(\lambda)$, with probability 
at least $1 - Ce^{-n/C}$, we have
\[
\left|\mathcal{SDP}(\bm{M}) - \mathcal{OPT}_k(\bm{M})\right| \leq \frac{C}{k}.
\]
\end{remark}

\begin{theorem}[OPT vs $\Phi$, Lemma 3.2 of \cite{MS16}]
\label{thm:OPT.Phi}
If $\bm{M} \in \mathbb{R}^{n \times n}$ satisfies
\[
\|\bm{M}\|_{\infty \to 2} := \max_{||\bm{v}||_\infty \leq 1} ||\bm{M}\bm{v}||_2 \leq L\sqrt{n},
\]
then for any $\beta > 0$, $k \in [n]$, and $\epsilon \in (0,1]$, there exists a constant $C$, such that
\[
\left|\frac{1}{n}\Phi(\beta, k; \bm{M}) - \frac{1}{n}\mathcal{OPT}_k(\bm{M})\right| 
\leq 2L\epsilon \sqrt{k} + \frac{k}{\beta}\log\frac{C}{\epsilon}.
\]
\end{theorem}
\begin{remark}
We can optimize the bound above over $\epsilon$ to obtain that when $\epsilon^*= \sqrt{k}/2\beta L
\in (0,1]$, we have
\[
\left|\frac{1}{n}\Phi(\beta, k; \bm{M}) - \frac{1}{n}\mathcal{OPT}_k(\bm{M})\right| 
\leq \frac{k}{\beta} \left(1 + \log\left(\frac{2C\beta L}{\sqrt{k}}\right)\right)
= \frac{k}{\beta} \log \frac{C\beta L}{\sqrt{k}}.
\]
\end{remark}
\begin{remark}
In particular, we can show that (Lemma A.3 of \cite{MS16}) for any graph $G$ with bounded degree, we have 
$\|\bm{A}^\mathrm{cen}_{G}\|_{\infty \to 2} \leq Cd\sqrt{n}$, and for any $\lambda \in \mathbb{R}$, 
we have $\|\bm{B}(\lambda)\|_{\infty \to 2} \leq (C + |\lambda|)\sqrt{n}$. so the above bound 
applies to $\bm{A}^\mathrm{cen}_{G}$ and $\bm{B}(\lambda)$ with $L = Cd$ and $L = C + |\lambda|$, respectively.
\end{remark}

They further showed the following concentration result for $\Phi$.
\begin{theorem}[Concentration of $\Phi$, Lemma A.1 of \cite{MS16}]
\label{thm:Phi.conc}
If $\bm{M} \in \mathbb{R}^{n \times n}$ is $\bm{A}^\mathrm{cen}_{G}$ for some graph $G$ with bounded degree,
or $\bm{B}(\lambda)$ for some $\lambda \in \mathbb{R}$, then for any $\beta > 0$ and $k \leq n$, 
for all $t>0$, we have
\[
\mathbb{P}\left(\left|\frac{1}{n}\Phi(\beta, k; \bm{M})
- \mathbb{E}\left(\frac{1}{n}\Phi(\beta, k; \bm{M})\right)\right| \geq t\right) \leq Ce^{-nt^2/C}.
\]
\end{theorem}

Therefore, the problem of approximating $\mathcal{SDP}(\bm{A}^\mathrm{cen}_{G})$ with
$\mathcal{SDP}(\bm{B}(\lambda))$ can be reduced to the problem of interpolating between
$\mathbb{E}(\Phi(\beta, k; \bm{A}^\mathrm{cen}_{G}))$ and $\mathbb{E}(\Phi(\beta, k; \bm{B}(\lambda)))$.
We will defer the details of the interpolation step to the next section, where we examine a more general
family of random graph models, the perturbed SBMs. As it turns out, the interpolation step will be the
most central step in extending the algorithm to the new setting.

\subsection{Lindeberg Interpolation}
\label{sec:sdp.interp}
The motivation of the interpolation step is the following Lindeberg interpolation lemma.
\begin{lemma}[Lindeberg interpolation, Theorem 1.1 of \cite{Cha06}]
\label{lem:lindeberg}
Let $F: \mathbb{R}^N \to \mathbb{R}$ be a thrice continuously differentiable function, and
let $\bm{X} = (X_1, \ldots, X_N)$ and $\bm{Z} = (Z_1, \ldots, Z_N)$ be two vectors of independent 
random variables satisfying $\mathbb{E}(X_i) = \mathbb{E}(Z_i)$ and $\mathbb{E}(X_i^2) = \mathbb{E}(Z_i^2)$ 
for all $i \in [N]$. Then, we have
\[
\left|\mathbb{E} \left( F(\bm{X}) - F(\bm{Z})\right)\right| \leq \frac{1}{6} 
\left(\max_{i \in [N]} \|\partial_i^3 F\|_\infty \right)
\left(\sum_{i=1}^{N} \mathbb{E}\left[|X_i|^3 + |Z_i|^3\right]\right).
\]
\end{lemma}
Thus, we construct a matrix $\bm{D}$, such that $\bm{D}$ is a Gaussian matrix with first and second 
moments matching those of $\bm{A}^\mathrm{cen}_{G} / \sqrt{d}$. Then, apply the above lemma to 
$F(\bm{M}) := \Phi(\beta, k; \bm{M})$ with $\bm{X} = \bm{A}^\mathrm{cen}_{G} / \sqrt{d}$ and 
$\bm{Z} = \bm{D}$, we can bound the difference between $F(\bm{A}^\mathrm{cen}_{G} / \sqrt{d})$ and $F(\bm{D})$.

First, we bound the third moments of the entries of $\bm{A}^\mathrm{cen}_{G} / \sqrt{d}$. Note that if 
$X \sim \operatorname{Ber}(p)$, then for $a \in [0,1]$, $\mathbb{E}[|X - a|^3] 
= (1-p)a^3 + p(1-a)^3 \leq a + p$. Since $\bm{A}^\mathrm{cen}_{G_{ij}} = \bm{A}_{G_{ij}} - d/n$, and
$\bm{A}_{G_{ij}} \sim \operatorname{Ber}\left(\frac{d}{n}(1 + \eta_{ij})\right)$,
we have 
\[
\sum_{i,j \in [n]} \mathbb{E} \left|\frac{1}{\sqrt{d}}\bm{A}^\mathrm{cen}_{G_{ij}}\right|^3 
\leq \frac{1}{d^{3/2}}\sum_{i,j \in [n]}\frac{d}{n}(2 + \eta_{ij})
\leq \frac{2n}{\sqrt{d}} + \frac{1}{n\sqrt{d}}\sum_{i,j \in [n]} h_{ij}
< \frac{3n}{\sqrt{d}},
\]
when $n$ is sufficiently large.

Next, we bound the third moments of the entries of $\bm{D}$. Note that if $X = a + bZ$, where
$Z \sim \mathcal{N}(0,1)$, then $\mathbb{E}[|X|^3] \leq 4(|a|^3 + |b|^3)$. Since
$\bm{D}_{ij} = \eta_{ij} \sqrt{d} / n  + \sqrt{p_{ij}(1-p_{ij})/d} \cdot Z_{ij}$, where $Z_{ij} \sim \mathcal{N}(0,1)$, we have
\begin{align*}
\sum_{i,j \in [n]} \mathbb{E} \left|\bm{D}_{ij}\right|^3 
&\leq 4\sum_{i,j \in [n]} \left(\left(\frac{\eta_{ij} \sqrt{d}}{n}\right)^3 
+ \left(\frac{p_{ij}(1-p_{ij})}{d}\right)^{3/2}\right)\\
&\leq \sum_{i,j \in [n]} \left(\frac{d^{3/2}}{n^3}\eta_{ij}^3 
+ \frac{1}{n^{3/2}}(1+\eta_{ij})^{3/2}\right)\\
&\leq \frac{d^{3/2}}{n} + 2\sqrt{2n}
< \frac{n}{\sqrt{d}},
\end{align*}
where we used the fact that $|\eta_{ij}| \leq 1$ and $d \ll n$.

Finally, Montanari and Sen proved that $\| \partial_{ij}^3 F\|_\infty \leq 6\beta^2$ 
for all $i, j \in [n]$ (line 112 of \cite{MS16}). Combining, we obtain
\[
\left|\mathbb{E} \left( F(\bm{A}^\mathrm{cen}_{G} / \sqrt{d}) - F(\bm{D})\right)\right|
\leq \frac{Cn\beta^2}{\sqrt{d}}
\]

\subsection{Coupling of Gaussian Matrices}
For simplicity, we write $\bm{B}(\lambda) = \bm{B}$. To bound the difference between $F(\bm{D})$ 
and $F(\bm{B}(\lambda))$, we first notice that, for all matrices $\bm{M}, \bm{M}' 
\in \mathbb{R}^{n \times n}$, we have 
\begin{align*}
F(\bm{M}) &= \frac{1}{\beta} \log \left(\int_{(\mathbb{S}^{k-1})^n} \exp\left(\beta \sum_{i,j=1}^n 
\bm{M}_{ij} \langle \bm{\sigma}_i, \bm{\sigma}_j \rangle\right) d\nu(\bm{\sigma})\right)\\
&= \frac{1}{\beta} \log \left(\int_{(\mathbb{S}^{k-1})^n} \exp\left(\beta \sum_{i,j=1}^n \bm{M}'_{ij}
\langle \bm{\sigma}_i, \bm{\sigma}_j \rangle + \beta \sum_{i,j=1}^n (\bm{M}_{ij} - \bm{M}'_{ij}) 
\langle \bm{\sigma}_i, \bm{\sigma}_j \rangle\right) d\nu(\bm{\sigma})\right)\\
&= F(\bm{M}') + \frac{1}{\beta} \log \left(\int_{(\mathbb{S}^{k-1})^n} \exp\left(\beta \sum_{i,j=1}^n 
(\bm{M}_{ij} - \bm{M}'_{ij}) \langle \bm{\sigma}_i, \bm{\sigma}_j \rangle\right) d\nu(\bm{\sigma})\right)\\
&= F(\bm{M}') + \frac{1}{\beta} \log \left(\int_{(\mathbb{S}^{k-1})^n} \exp\left(\beta \operatorname{Tr} 
(\bm{\Sigma} (\bm{M} - \bm{M}') \bm{\Sigma}^\top) \right) d\nu(\bm{\sigma})\right)\\
&\leq F(\bm{M}') + \frac{1}{\beta} \log \left(\int_{(\mathbb{S}^{k-1})^n} \exp\left(\beta 
\|\bm{M} - \bm{M}'\|_{\mathrm{op}} \operatorname{Tr} (\bm{\Sigma}^\top \bm{\Sigma}) \right) 
d\nu(\bm{\sigma})\right)\\
&= F(\bm{M}') + \frac{1}{\beta} \log \left(\int_{(\mathbb{S}^{k-1})^n} \exp\left(n\beta 
\|\bm{M} - \bm{M}'\|_{\mathrm{op}}\right) d\nu(\bm{\sigma})\right)\\
&= F(\bm{M}') + n \|\bm{M} - \bm{M}'\|_{\mathrm{op}}.
\end{align*}
where we wrote $\bm{\Sigma} := (\bm{\sigma}_1, \dots, \bm{\sigma}_n) \in (\mathbb{S}^{k-1})^n$,
and used the fact that for $\bm{A}$ symmetric and $\bm{B} \succeq 0$, since $\bm{A} \preceq 
\|\bm{A}\|_{\mathrm{op}} \bm{I}$, we have
\[
\operatorname{Tr}(\bm{A}\bm{B}) \leq \operatorname{Tr}(\|\bm{A}\|_{\mathrm{op}} \bm{I} \bm{B})
\leq \|\bm{A}\|_{\mathrm{op}} \operatorname{Tr}(\bm{B}).
\]
Hence,
\[
\left| \frac{1}{n}F(\bm{M}) - \frac{1}{n}F(\bm{M}')\right| 
\leq \|\bm{M} - \bm{M}'\|_{\mathrm{op}}.
\]
Therefore,
\begin{align*}
\frac{1}{n} \left|\mathbb{E}F(\bm{D}) - \mathbb{E}F(\bm{B})\right|
&\leq \frac{1}{n} \mathbb{E} \left|F(\bm{D}) - F(\bm{B})\right|\\
&\leq \mathbb{E} \|\bm{D} - \bm{B}\|_{\mathrm{op}}.
\end{align*}
Thus, we want to find a bound on $\mathbb{E} \|\bm{D} - \bm{B}\|_{\mathrm{op}}$. To do this,
we will use a coupling argument. Note that the diagonal entries of $\bm{D}$ and $\bm{B}$ are zero, 
and the entries are symmetric, so we only need to consider the entries above the diagonal.
For simplicity, we fix a realization of $\bm{H}$. Writing $V_{ij} = n p_{ij} (1 - p_{ij}) / d$, 
note that for all $i, j \in [n]$, $i < j$, we have
\[
V_{ij} = (1 + \eta_{ij})\left(1 - \frac{d}{n}(1 + \eta_{ij})\right),
\]
\[
\bm{D}_{ij} \sim \mathcal{N}\left(
\frac{\lambda}{n} x_i x_j + \frac{\sqrt{d}}{n} h_{ij}, \ \frac{1}{n}V_{ij}\right),
\]
\[
\bm{B}_{ij} \sim \mathcal{N}\left(
\frac{\lambda}{n} x_i x_j, \ \frac{1}{n}\right).
\]
 
Thus, we can couple $\bm{D}$ and $\bm{B}$ such that
\[
\bm{D} = \frac{\lambda}{n} \bm{x} \bm{x}^\top + \frac{\sqrt{d}}{n} \bm{H} 
+ \bm{Z_0} + \bm{Z_D},
\]
\[
\bm{B} = \frac{\lambda}{n} \bm{x} \bm{x}^\top + \bm{Z_0} + \bm{Z_B},
\]
where $\bm{Z_0}, \bm{Z_D}, \bm{Z_B}$ are Gaussian matrices with mean-zero, independent entries.

Writing $V_- = \min_{i,j} \left(-\|- V_{ij}\|_\infty\right)$, the variances of the entries are given by
\[
\operatorname{Var}(\bm{Z_0}{}_{,ij}\mid \bm{H}) = \begin{cases}
\frac{1}{n}V_- & \text{if } i \neq j,\\
0 & \text{if } i = j,
\end{cases}
\]
\[
\operatorname{Var}(\bm{Z_D}{}_{,ij}\mid \bm{H}) = \begin{cases}
\frac{1}{n}(V_{ij} - V_-) & \text{if } i \neq j,\\
0 & \text{if } i = j,
\end{cases}
\]
\[
\operatorname{Var}(\bm{Z_B}{}_{,ij}\mid \bm{H}) = \begin{cases}
\frac{1}{n}(1 - V_-) & \text{if } i \neq j,\\
0 & \text{if } i = j,
\end{cases}
\]
Under this coupling, by the triangle inequality, we have
\[
\mathbb{E} \| \bm{D} - \bm{B}\|_{\mathrm{op}} =
\mathbb{E} \left\|\frac{\sqrt{d}}{n} \bm{H} + \bm{Z_D} - \bm{Z_B}\right\|_{\mathrm{op}}
\leq \frac{\sqrt{d}}{n} \mathbb{E} \|\bm{H}\|_{\mathrm{op}} + \mathbb{E} \|\bm{Z_D}\|_{\mathrm{op}} 
+ \mathbb{E} \|\bm{Z_B}\|_{\mathrm{op}}.
\]
We then use the following result for operator norm of centered Gaussian matrices.
\begin{theorem}[Operator norm of Gaussian matrices, Theorem 1.1 of \cite{Ban16}]
\label{thm:gaussian.op}
Let $\bm{X}$ be an $n \times n$ symmetric matrix with $\bm{X}_{ij} = g_{ij}b_{ij}$ for $i \geq j$, 
where $g_{ij}$ are independent standard Gaussian variables, and $b_{ij}$ are given scalars. 
Then, writing 
\[
\sigma^*_\text{col} = \max_{j \in [n]} \sqrt{\sum_{i=1}^n b_{ij}^2}, \quad
\sigma^*_\text{elt} = \max_{i,j \in [n]} |b_{ij}|,
\]
for every $s\in (0,1/2]$, we have
\[
\mathbb{E} \| \bm{X} \|_{\mathrm{op}} \leq (1+s) \left(2\sigma^*_\text{col} 
+ \frac{6}{\sqrt{\log(1+s)}} \sigma^*_\text{elt} \sqrt{\log n}\right)
\]
\end{theorem}
In our case, writing $V_+ = \max_{i,j} \|V_{ij}\|_\infty$, for all $i, j \in [n]$, we have
\[ 
\operatorname{Var}(\bm{Z_D}{}_{,ij} \mid \bm{H}),\ \operatorname{Var}(\bm{Z_B}{}_{,ij} \mid \bm{H}) 
\leq \frac{1}{n}(V_+ - V_-).
\]
Thus, writing $\delta V := V_+ - V_-$, we have $\sigma^*_\text{col} \leq \sqrt{\delta V}$, 
$\sigma^*_\text{elt} \leq \sqrt{\delta V / n}$. Applying the above theorem,
\[
\mathbb{E} \left(\|\bm{Z_D}\|_{\mathrm{op}} \mid \bm{H}\right), 
\ \mathbb{E} \left(\|\bm{Z_B}\|_{\mathrm{op}} \mid \bm{H}\right)
\leq (1+s) \left(2 + \frac{6}{\sqrt{\log(1+s)}} \sqrt{\frac{\log n}{n}} \right)\sqrt{\delta V}
\leq C \sqrt{\delta V},
\]
when $n$ is sufficiently large. Thus, by the tower law of expectation, we have
\[
\mathbb{E} \| \bm{Z_D}\|_{\mathrm{op}}, \ \mathbb{E} \| \bm{Z_B}\|_{\mathrm{op}}
\leq C \sqrt{\delta V}.
\]

Recall that $V_{ij} = np_{ij}(1-p_{ij})/d$, and $p_{ij} \ll 1/2$, so $V_{ij}$ is increasing in $p_{ij}$,
and thus increasing in $\eta_{ij}$. Since $|\eta_{ij}| \leq \epsilon/2 + \|h\|_\infty$, we have 
\begin{align*}
\delta V = V_+ - V_- &\leq \left(1 + \frac{\epsilon}{2} + \|h\|_\infty\right)
\left(1 - \frac{d}{n}\left(1 + \frac{\epsilon}{2} + \|h\|_\infty\right)\right)\\
&- \left(1 - \frac{\epsilon}{2} - \|h\|_\infty\right)
\left(1 - \frac{d}{n}\left(1 - \frac{\epsilon}{2} - \|h\|_\infty\right)\right)\\
&\leq \left(1 - \frac{d}{n}\right)(\epsilon + 2\|h\|_\infty)\\
&<\epsilon + 2\|h\|_\infty
< C d^{-2\alpha},
\end{align*}
where we used the fact that $\epsilon = \lambda d^{-1/2} = O(d^{-1/2})$ and $\|h\|_\infty = O(d^{-2\alpha})$.

Therefore, we have
\begin{align*}
\mathbb{E} \| \bm{D} - \bm{B}\|_{\mathrm{op}}
&\leq \frac{\sqrt{d}}{n} \mathbb{E} \|\bm{H}\|_{\mathrm{op}} + \mathbb{E} \|\bm{Z_D}\|_{\mathrm{op}}
+ \mathbb{E} \|\bm{Z_B}\|_{\mathrm{op}}\\
&\leq \frac{\sqrt{d}}{n} \left(C d^{-1/2-\alpha}\right) + C \sqrt{\delta V} \\
&= Cd^{-\alpha}.
\end{align*}
Combining with the bound from the Lindeberg interpolation step, we have
\[
\frac{1}{n} \left| \mathbb{E}F\left(\frac{1}{\sqrt{d}}\bm{A}^\mathrm{cen}_{G}\right) 
- \mathbb{E}F(\bm{B})\right| \leq C\left(\frac{\beta^2}{\sqrt{d}} + d^{-\alpha}\right).
\]

\subsection{Combining all the pieces}
With the above bound between $\mathbb{E}F$ values, we first use the concentration of $\Phi$ from
\cref{thm:Phi.conc} to show that taking $t = O(d^{-1/2})$, with probability at least $1 - Ce^{-n/C}$,
we have
\[
\frac{1}{n} \left| \Phi\left(\beta, k; \frac{1}{\sqrt{d}}\bm{A}^\mathrm{cen}_{G}\right) 
- \Phi(\beta, k; \bm{B})\right| \leq C\left(\frac{\beta^2}{\sqrt{d}} + d^{-\alpha}\right).
\]
Then, \cref{thm:OPT.Phi} allows us to relate $\Phi$ and $\mathcal{OPT}_k$, which gives
\begin{align*}
\frac{1}{n} \left|\mathcal{OPT}_k\left(\frac{1}{\sqrt{d}}\bm{A}^\mathrm{cen}_{G}\right)
- \mathcal{OPT}_k(\bm{B})\right| &\leq \frac{k}{\beta} \log \frac{C\beta d}{\sqrt{k}} 
+ \frac{k}{\beta} \log \frac{C\beta \lambda}{\sqrt{k}} + C\left(\frac{\beta^2}{\sqrt{d}} + d^{-\alpha}\right)\\
&\leq \frac{k}{\beta} \log \frac{C\beta d\lambda}{k} + C\frac{\beta^2}{\sqrt{d}} + C d^{-\alpha}.
\end{align*}
Note that this holds for all $\beta \in \mathbb{R}$, so taking $\beta = d^{1/6} k^{1/3}$, we have
\begin{align*}
\frac{1}{n} \left|\mathcal{OPT}_k\left(\frac{1}{\sqrt{d}}\bm{A}^\mathrm{cen}_{G}\right)
- \mathcal{OPT}_k(\bm{B})\right| &\leq C d^{-1/6} k^{2/3} \log(Cd^{7/6} k^{-2/3} \lambda) + C d^{-\alpha}\\
&\leq C d^{-1/6} k^{2/3} \log(Cd^{7/6} \lambda) + C d^{-\alpha},
\end{align*}
where we dropped the $k^{-2/3}$ term in the log since $k$ is a positive integer.

Finally, \cref{thm:SDP.OPT} links $\mathcal{OPT}_k$ and $\mathcal{SDP}$, which gives
\[
\frac{1}{n} \left|\mathcal{SDP}\left(\frac{1}{\sqrt{d}}\bm{A}^\mathrm{cen}_{G}\right)
- \mathcal{SDP}(\bm{B})\right| \leq \frac{C}{k} + C d^{-1/6} k^{2/3} \log(Cd^{7/6} \lambda) + C d^{-\alpha}.
\]
Taking $k = d^{1/10}$, we have
\[
\frac{1}{n} \left|\mathcal{SDP}\left(\frac{1}{\sqrt{d}}\bm{A}^\mathrm{cen}_{G}\right)
- \mathcal{SDP}(\bm{B})\right| \leq C \frac{\log(Cd^{7/6}\lambda)}{d^{1/10}} + C d^{-\alpha}.
\]
Crucially, the above bound goes to zero as $d \to \infty$, which allows us to conclude that the same
algorithm still works for the perturbed SBM, proving \cref{thm:perturbed.SBM}. \hfill $\blacksquare$

\section{Conclusion and Future Directions}
\label{sec:conclusion}
In this essay, we studied several notions of robustness in community detection, with a particular
focus on SDP-based algorithms for weak recovery in sparse stochastic block models. We first reviewed
existing robustness models, including monotone adversaries, edge perturbations, node perturbations,
and degree heterogeneity. We then focused on the algorithm of Montanari and Sen, whose SDP relaxation
achieves weak recovery almost at the Kesten-Stigum threshold. The main contribution of this essay was
to show that the same algorithm is robust under a more general perturbation model, where the edge
probabilities themselves are allowed to fluctuate by a matrix $h$. The resulting condition is not
simply a bound on the number of modified edges: rather, it depends on both the entrywise size of the
perturbation and its operator norm. This distinction captures the key phenomenon that unstructured
perturbations can cancel spectrally and hence be much less harmful than structured, low-rank
perturbations of with similar expected edge modification counts.

There are several natural directions for further work. First, it would be interesting to weaken the
entrywise condition on $h$, since the present theorem excludes some perturbations that may still
be benign after applying concentration bounds. Second, one could ask whether analogous $h$-style
robustness results hold for other community detection algorithms, especially those not based on the
specific SDP interpolation framework used here. Finally, it would be valuable to understand whether
weak recovery guarantees for degree-corrected stochastic block models can be made independent of the
node degree parameters after imposing additional constraints, and whether this gives a more intrinsic 
bridge between degree heterogeneity and node corruption, perhaps mirroring the relationship between
$h$-style perturbations and edge corruption.
